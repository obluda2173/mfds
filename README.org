* About This Repository
Welcome to the unofficial open-source study hub for the *Bachelor of Mathematics for Data Science* (First Cohort).

This repository is a centralized workspace containing notes, code solutions, exercises, and study guides. It transforms scattered lecture knowledge into a structured, searchable, and reproducible archive.

*Goal:* To create a "Single Source" for our studies, helping us handle the coursework and succeed in the exams.

/Disclaimer: This is a student-run project. It is not officially affiliated with the University of Vienna. No copyrighted university materials are hosted here—only original notes, code, and links to official resources./

* The Workflow (Why Org-Mode?)
This repository is built using *Doom Emacs* + *Org-Mode*. While you don't need Emacs to read the notes (GitHub renders them), this stack was chosen for specific strategic reasons:

1.  *Organization at Scale:* Centralized, structured notes + git = traceability. No more lost files.
2.  *Reproducibility:* Code (Python, C++, Julia) is embedded directly in the notes. We can run and verify math immediately.
3.  *Searchability:* `grep` your entire degree. Finding a specific theorem or code snippet takes milliseconds.
4.  *LLM-Ready:* Plain text files are the perfect input for LLMs. We use AI to automate the monotonous work, leaving the complex thinking to us.
5.  *Exam Alignment:* The structure mirrors our problem sets, creating a near-optimal architecture for exam prep.
