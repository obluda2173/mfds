* MFDS — A Workspace Built on a Thesis

Most students treat a degree as four years of disposable notebooks. They hand-copy equations they'll never read again, stack PDFs in folders they'll never open, and call it studying. Then they're surprised when nothing compounds.

I don't.

This repository is my degree as a single, versioned, executable artifact. Every theorem I encounter gets encoded, not drawn. Every script I write survives the semester it was written in. Every note is plain text the moment it exists, because plain text is the only format that doesn't rot.

Org-mode. LaTeX. Julia / Python / C++ under Org-babel. Doom Emacs. Git. An LLM ruleset that enforces how machine-extracted material enters the graph.

You won't find polish here. You'll find a method. The method is the point.
