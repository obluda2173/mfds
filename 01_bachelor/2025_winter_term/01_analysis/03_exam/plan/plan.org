*Theme: "The Algorithm of Convergence"*

- [ ] **ps_01, Problem 4**: Euler number sequence $(1 + \frac{1}{n+1})^{n+1}$.
  - *Focus:* Boundedness and Binomial Theorem.
- [X] **ps_00, Problem 3**: Limits at 0.

** Day 3: Continuity
*Theme: "Breaking and Connecting Functions"*
- [X] **ps_02, Problem 2**: Continuity of Composition.
  - *Problem:* $f(x) = \frac{1}{x^2+1}$ and $g(x) = \exp(|x|)$.
- [X] **ps_02, Problem 3**: Continuity by Cases ($x=0$).
  - *Problem:* $\exp(-1/x)$ for $x \neq 0$. Check limits as $x \to 0^+$ vs $x \to 0^-$.
- [X] **ps_02, Problem 1**: Dirichlet-style function.
  - *Problem:* $f(x)=1$ if $x^2 > 2$, else $0$ on $\mathbb{Q}$.
- [X] **ps_03, Problem 1**: ReLU Function.
  - *Focus:* $\epsilon-\delta$ definition at the "kink" ($x=0$).

** Day 4: Exponentials, Logarithms & Interpolation
*Theme: "Constructing and Approximating"*
- [X] **ps_03, Problem 6**: Truncated Series.
  - *Task:* Implement/Write approximations for $\sin(x)$ and $\cos(x)$.
- [X] **ps_03, Problem 4**: Linear Approximation.

- [X] **ps_03, Problem 2**: Softplus activation $g(x)=\log(1+e^x)$.

- [X] **ps_02, Problem 5**: Sea-level rise model.
