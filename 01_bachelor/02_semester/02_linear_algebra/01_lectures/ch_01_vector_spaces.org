* Definition 1.1 Binary operation $+$ on a set $E$

A binary operation on set $E$ is a mapping

\begin{align*}
+ : E \times E &\rightarrow E \\
(x, y) &\mapsto x + y \\
\end{align*}

which produces an element $E$ out of two elements from $E$.

* Definition 1.3 Commutative Group

We say that $(E, +)$ is a group when $E$ is a set, and $+$ is a binary operation on $E$ satisfying:

1. Associativity: $(x + y) + z = x + (y + z), \quad \forall x, y, z \in E$
2. Identity element: $\exists 0_{E} : x + 0_{E} = x$
3. Inverse element: $\forall x \in E, \exists \tilde{x} \in E : x + \tilde{X} = 0_{E}$
4. Commutativity: $x + y = y + x$
