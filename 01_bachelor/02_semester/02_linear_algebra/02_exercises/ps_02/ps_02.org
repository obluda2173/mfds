* Problem 1.24 :DONE:
** Task
#+begin_quote
(Reverse Triangle Inequality). From the triangle inequality, derive the /reverse triangle inequality/, namely

$| \|x\| - \|y\| | \leq \|x - y\|$, $\forall (x, y) \in \mathbb{K}$

where $\mathbb{K} = \mathbb{R}^n$ or $\mathbb{C}^n$ and $\|\cdot\|$ is any norm in $\mathbb{R}^n$.
#+end_quote

** Solution
First, rewrite $x = (x - y) + y$ and apply the triangle inequality:

\begin{align*}
\|x\| &= \|(x-y) + y\| \\
       &\le \|x-y\| + \|y\|
\end{align*}

which gives:

$$\|x\| - \|y\| \le \|x-y\| \tag{1}$$

Similarly, rewrite $y = (y - x) + x$ and apply the triangle inequality:

\begin{align*}
\|y\| &= \|(y-x) + x\| \\
       &\le \|y-x\| + \|x\|
\end{align*}

By homogeneity, $\|y - x\| = \|{-(x-y)}\| = |{-1}| \cdot \|x-y\| = \|x-y\|$, so:

$$\|y\| - \|x\| \le \|x-y\| \tag{2}$$

Combining (1) and (2): since both $\|x\|-\|y\| \le \|x-y\|$ and $-(\|x\|-\|y\|) \le \|x-y\|$, we have:

$$\bigl|\|x\| - \|y\|\bigr| \le \|x-y\|$$

* Problem 1.29 :DONE:
** Task
#+begin_quote
(Vector space) On $\mathbb{R}^2$ we define the following operations

$(x_1, x_2) \oplus (y_1, y_2) = (x_1 + y_1, x_2 + y_2)$, $\lambda \odot (x_1, x_2) = (\lambda x_1, 0)$.

Which vector space axioms does $(\mathbb{R}^2, \oplus, \odot)$ satisfy? Is it a vector space?
#+end_quote

** Solution
1. *Associativity of addition*
   $(x \oplus y) \oplus z = (x_1+y_1+z_1,\, x_2+y_2+z_2) = x \oplus (y \oplus z)$.

2. *Commutativity of addition*
   $x \oplus y = (x_1+y_1,\, x_2+y_2) = y \oplus x$.

3. *Additive identity*
   $(x_1, x_2) \oplus (0, 0) = (x_1, x_2)$.

4. *Additive inverse*
   $(x_1, x_2) \oplus (-x_1, -x_2) = (0, 0)$.

5. *Distributivity over scalar addition*
   Both sides yield $((\lambda+\mu)x_1,\, 0)$.

6. *Distributivity over vector addition*
   Both sides yield $(\lambda(x_1+y_1),\, 0)$.

7. *Compatibility of scalar multiplication*
   Both sides yield $(\lambda\mu\, x_1,\, 0)$.

8. *Scalar multiplicative identity*
   $1 \odot (x_1, x_2) = (x_1, 0) \neq (x_1, x_2)$ when $x_2 \neq 0$.

*Conclusion:* $(\mathbb{R}^2, \oplus, \odot)$ is not a vector space.

Axioms 1-7 hold, but axiom 8 fails: scalar multiplication by $1$ zeroes out the second component.

* Problem 1.33 :DONE:
** Task
#+begin_quote
(Bases and regular matrices) Let $\mathcal{U} = \{u_1, \ldots, u_n\}$ be a basis of a vector space $\mathbb{E}$. Let $A \in \mathbb{K}^{n \times n}$ and vectors $v_j = \sum_{i=1}^{n} A_{ij} u_i$. Prove that $\mathcal{V} = \{v_1, \ldots, v_n\}$ is also a basis if and only if $A$ is invertible.
#+end_quote

** Solution

- ($\Rightarrow$)

  If $\mathcal{V}$ is basis, every $u_i$ can be expressed as a linear combination of $v_j$:

  $$u_i = \sum_{j=1}^n B_{ji} v_j$$

  for some matrix $B \in \mathbb{K}^{n \times n}$, where $B_{ji}$ is the coefficient of $v_j$ in the expansion of $u_i$.

  Now substitute $v_j = \sum_{k=1}^n A_{kj}u_k$:

  $$u_i = \sum_{j=1}^n B_{ji} \sum_{k=1}^n A_{kj}u_k = \sum_{k=1}^n \left( \sum_{j=1}^n A_{kj} B_{ji} \right) u_k$$

  The inner sum is exactly the $(k,i)$ entry product $AB$.

  $$u_i = \sum_{k=1}^n \left( AB \right)_{ki} u_k$$

  Since $\mathcal{U}$ is a basis, the representation of $u_i$ in terms of $u_1, \cdots, u_n$ is unique, and it is simply $u_i = 0 \cdot u_1 + \cdots + 1 \cdot u_i + \cdots + 0 \cdot u_n$. Therefore,

  $$(AB)_{ki} = \delta_{ki} = \begin{cases} 1 & k = i \\ 0 & k \neq i \end{cases}$$

  This gives $AB = I$. Since $A$ is a square matrix with a right inverse, it is invertible.

- ($\Leftarrow$)

  If $A$ is invertible, we need to show $\mathcal{V}$ is a basis. Since $|\mathcal{V}| = n = \dim \mathbb{E}$, it suffices to show $\mathcal{V}$ is linearly independent.

  Suppose $\sum_{j=1}^n x_j v_j = 0$ for some scalars $x_1, \dots, x_n$.

  Substitute $v_j = \sum_{i=1}^n A_{ij} u_i$:

  $$\sum_{j=1}^n x_j \sum_{i=1}^n A_{ij} u_i = \sum_{i=1}^n \left( \sum_{j=1}^n A_{ij} x_j \right) u_i = \sum_{i=1}^n (Ax)_i\, u_i = 0$$

  Since $\mathcal{U}$ is a basis, the $u_i$ are linearly independent, so every coefficient must vanish:

  $$(Ax)_i = 0 \quad \text{for all } i = 1, \dots, n$$

  This means $Ax = 0$. Since $A$ is invertible, $x = A^{-1} \cdot 0 = 0$, i.e. $x_1 = \cdots = x_n = 0$.

  Therefore $\mathcal{V}$ is linearly independent, and since it has $n$ vectors in an $n$ dimensional space, $\mathcal{V}$ is a basis.

* Problem 1.37 :DONE:
** Task
#+begin_quote
(Direct sum of triangular matrices) Let $\mathbb{E} = \mathbb{K}^{n \times n}$, we define $\mathbb{F}_1, \mathbb{F}_2 \subset \mathbb{E}$ as the subspaces of upper and lower triangular matrices, respectively. Determine whether $\mathbb{E} = \mathbb{F}_1 \oplus \mathbb{F}_2$. Does the result vary if we replace $\mathbb{F}_1$ by $\mathbb{F}_1'$ the subspace of 1-stricly triangular matrices (as defined in ex. 1.28)?
#+end_quote

** Solution
*** (1)

*Proof.*

We determine whether $\mathbb{E} = \mathbb{F}_1 \oplus \mathbb{F}_2$ using the Dimension Theorem for Sums of Subspaces:

$$\dim(\mathbb{F}_1 + \mathbb{F}_2) = \dim(\mathbb{F}_1) + \dim(\mathbb{F}_2) - \dim(\mathbb{F}_1 \cap \mathbb{F}_2).$$

A matrix that is simultaneously upper and lower triangular can only have nonzero entries on the diagonal, so

$\mathbb{F}_1 \cap \mathbb{F}_2 = \mathcal{D}_n$ and $\dim(\mathbb{F}_1 \cap \mathbb{F}_2) = n$.

Plugging in,

\begin{align*}
\dim(\mathbb{F}_1 + \mathbb{F}_2)
&= \frac{n(n+1)}{2} + \frac{n(n+1)}{2} - n \\
&= n^2 + n - n \\
&= n^2 = \dim(\mathbb{E}).
\end{align*}

So $\mathbb{F}_1 + \mathbb{F}_2 = \mathbb{E}$. However, for the sum to be direct we require $\dim(\mathbb{F}_1 \cap \mathbb{F}_2) = 0$, and we found it equals $n \geq 1$.

Therefore $\mathbb{E} \neq \mathbb{F}_1 \oplus \mathbb{F}_2$.

*** (2)

*Proof.*

We determine whether $\mathbb{E} = \mathbb{F}_1' \oplus \mathbb{F}_2$ by checking the two conditions for a direct sum.

A matrix $A \in \mathbb{F}_1' \cap \mathbb{F}_2$ satisfies

$a_{ij} = 0$ for $i \geq j$ (strictly upper triangular) and

$a_{ij} = 0$ for $i < j$ (lower triangular). Together these force

$a_{ij} = 0$ for all $i, j$, so

$\mathbb{F}_1' \cap \mathbb{F}_2 = \{0\}$.

It remains to verify that $\mathbb{F}_1' + \mathbb{F}_2 = \mathbb{E}$. By the Dimension Theorem for Sums of Subspaces,

\begin{align*}
\dim(\mathbb{F}_1' + \mathbb{F}_2)
&= \dim(\mathbb{F}_1') + \dim(\mathbb{F}_2) - \dim(\mathbb{F}_1' \cap \mathbb{F}_2) \\
&= \frac{n(n-1)}{2} + \frac{n(n+1)}{2} - 0 \\
&= \frac{n^2 - n + n^2 + n}{2} \\
&= n^2 = \dim(\mathbb{E}).
\end{align*}

Both conditions are satisfied, so $\mathbb{E} = \mathbb{F}_1' \oplus \mathbb{F}_2$.
