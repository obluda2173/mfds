* C.1 Counting

** Rules of sum and product

- *Rule of sum:* disjoint sets $A, B$ → $|A \cup B| = |A| + |B|$
- *Rule of product:* ordered pairs from $A \times B$ → $|A \times B| = |A| \cdot |B|$

** Strings

- *String* over finite set $S$: sequence of elements of $S$; length-$k$ string = *$k$ string*
- *Substring*: contiguous subsequence; *$k$ substring*: substring of length $k$
- $k$ string over $S$ ↔ element of $S^k$, so $|S|^k$ total $k$ strings
- Binary $k$ strings: $2^k$

** Permutations

- *Permutation* of $S$: ordered sequence of all elements, each exactly once → $n!$
- *$k$ permutation*: ordered sequence of $k$ distinct elements from $n$ set → $\frac{n!}{(n-k)!}$

** Combinations

- *$k$ combination*: $k$ subset (order irrelevant)
- Count: $k$ permutations / $k!$ (each subset counted $k!$ times) → $\frac{n!}{k!(n-k)!}$

** Binomial coefficients

- $\binom{n}{k} = \frac{n!}{k!(n-k)!}$
- Symmetry: $\binom{n}{k} = \binom{n}{n-k}$
- *Binomial theorem:* $(x+y)^n = \sum_{k=0}^{n} \binom{n}{k} x^k y^{n-k}$
- Special case $x=y=1$: $2^n = \sum_{k=0}^{n} \binom{n}{k}$ (counting binary $n$ strings by number of 1s)

** Binomial bounds

- Lower: $\binom{n}{k} \geq \left(\frac{n}{k}\right)^k$
- Upper (Stirling-based, using $k! \geq (k/e)^k$): $\binom{n}{k} \leq \frac{n^k}{k!} \leq \left(\frac{en}{k}\right)^k$
- Upper (entropy): $\binom{n}{\lambda n} \leq 2^{n H(\lambda)}$ where $H(\lambda) = -\lambda \lg \lambda - (1-\lambda) \lg(1-\lambda)$ is the binary entropy function
  - Derived from $\binom{n}{k} \leq \frac{n^n}{k^k (n-k)^{n-k}}$ with $k = \lambda n$
  - Conventions: $0^0 = 1$, $0 \lg 0 = 0$, so $H(0) = H(1) = 0$

* C.2 Probability

** Sample space and events

- *Sample space* $S$: set of outcomes (elementary events)
- *Event*: subset of $S$
- *Certain event*: $S$; *null event*: $\emptyset$
- *Mutually exclusive*: $A \cap B = \emptyset$

** Axioms of probability

1. $\Pr\{A\} \geq 0$
2. $\Pr\{S\} = 1$
3. $\Pr\{A \cup B\} = \Pr\{A\} + \Pr\{B\}$ for mutually exclusive $A, B$; extends to countable unions

** Immediate consequences

- $\Pr\{\emptyset\} = 0$
- $A \subseteq B \Rightarrow \Pr\{A\} \leq \Pr\{B\}$
- *Complement*: $\bar{A} = S - A$, $\Pr\{\bar{A}\} = 1 - \Pr\{A\}$
- *Inclusion-exclusion*: $\Pr\{A \cup B\} = \Pr\{A\} + \Pr\{B\} - \Pr\{A \cap B\}$
- *Union bound*: $\Pr\{A \cup B\} \leq \Pr\{A\} + \Pr\{B\}$

** Discrete probability distributions

- Discrete: finite or countably infinite $S$
- $\Pr\{A\} = \sum_{s \in A} \Pr\{s\}$
- *Uniform distribution*: $\Pr\{s\} = 1/|S|$ for all $s$
- Coin flips: $S = \{H,T\}^n$, $|S| = 2^n$; event "exactly $k$ heads" has probability $\binom{n}{k}/2^n$

** Continuous uniform distribution

- Defined over $[a, b]$; $\Pr\{[c,d]\} = \frac{d - c}{b - a}$
- Single point has probability 0
- Events: subsets obtainable by countable unions of intervals (plus more complicated sets)

** Conditional probability

- $\Pr\{A \mid B\} = \frac{\Pr\{A \cap B\}}{\Pr\{B\}}$, requires $\Pr\{B\} \neq 0$

** Independence

- *Independent*: $\Pr\{A \cap B\} = \Pr\{A\}\Pr\{B\}$; equivalently $\Pr\{A \mid B\} = \Pr\{A\}$
- *Pairwise independent*: $\Pr\{A_i \cap A_j\} = \Pr\{A_i\}\Pr\{A_j\}$ for all $i < j$
- *(Mutual) independence*: $\Pr\{A_{i_1} \cap \cdots \cap A_{i_k}\} = \prod \Pr\{A_{i_j}\}$ for every $k$ subset, $2 \leq k \leq n$
- Pairwise independence $\not\Rightarrow$ mutual independence

** Bayes's theorem

- From $A \cap B = B \cap A$: $\Pr\{A \cap B\} = \Pr\{B\}\Pr\{A \mid B\} = \Pr\{A\}\Pr\{B \mid A\}$
- $\Pr\{A \mid B\} = \frac{\Pr\{A\}\Pr\{B \mid A\}}{\Pr\{B\}}$
- Expanded form: $\Pr\{A \mid B\} = \frac{\Pr\{A\}\Pr\{B \mid A\}}{\Pr\{A\}\Pr\{B \mid A\} + \Pr\{\bar{A}\}\Pr\{B \mid \bar{A}\}}$

* C.3 Discrete random variables

** Discrete random variables

- A *(discrete) random variable* $X$ is a function from a finite or countably infinite sample space $S$ to $\mathbb{R}$.

- The event $X = x$ is defined as $\{s \in S : X(s) = x\}$, so $\Pr\{X = x\} = \sum_{s \in S : X(s) = x} \Pr\{s\}$.

- The *probability density function* (pdf) of $X$ is $f(x) = \Pr\{X = x\}$.

  - From axioms: $\Pr\{X = x\} \geq 0$ and $\sum_x \Pr\{X = x\} = 1$.

- Example: rolling two fair dice, $X$ = maximum of the two values. $\Pr\{X = 3\} = 5/36$ since $X$ assigns 3 to the outcomes $(1,3), (2,3), (3,3), (3,2), (3,1)$.

** Joint and marginal distributions

- For two random variables $X, Y$ on the same sample space, the *joint probability density function* is $f(x, y) = \Pr\{X = x \text{ and } Y = y\}$.

- Marginals recovered by summing out:

  \[ \Pr\{Y = y\} = \sum_x \Pr\{X = x \text{ and } Y = y\} \]
  \[ \Pr\{X = x\} = \sum_y \Pr\{X = x \text{ and } Y = y\} \]
- Conditional probability for random variables (from definition C.16):

  \[ \Pr\{X = x \mid Y = y\} = \frac{\Pr\{X = x \text{ and } Y = y\}}{\Pr\{Y = y\}} \]

- Two random variables $X$ and $Y$ are *independent* if for all $x$ and $y$, $\Pr\{X = x \text{ and } Y = y\} = \Pr\{X = x\}\Pr\{Y = y\}$.

** Expected value of a random variable

- The *expected value* (or *expectation*, *mean*) of a discrete random variable $X$:

  \[ \mathrm{E}[X] = \sum_x x \cdot \Pr\{X = x\} \tag{C.23} \]

  well defined when the sum is finite or converges absolutely.

- Sometimes denoted $\mu_X$ or just $\mu$.

- Example: flip two fair coins, earn $3$ per head, lose $2$ per tail. $\mathrm{E}[X] = 6 \cdot (1/4) + 1 \cdot (1/2) - 4 \cdot (1/4) = 1$.

** Linearity of expectation

- *Linearity of expectation*: for any two random variables $X, Y$,

  \[ \mathrm{E}[X + Y] = \mathrm{E}[X] + \mathrm{E}[Y] \tag{C.24} \]
  whenever both expectations are defined. Holds even when $X$ and $Y$ are not independent.

- Extends to finite and absolutely convergent summations.

- Key property enabling probabilistic analyses via indicator random variables (see §5.2).

- For any random variable $X$ and function $g$, the random variable $g(X)$ has expectation $\mathrm{E}[g(X)] = \sum_x g(x) \cdot \Pr\{X = x\}$, provided this is defined.

- Scalar multiplication: for any constant $a$,

  \[ \mathrm{E}[aX] = a\,\mathrm{E}[X] \tag{C.25} \]

- Combined linearity: for any random variables $X, Y$ and constant $a$,

  \[ \mathrm{E}[aX + Y] = a\,\mathrm{E}[X] + \mathrm{E}[Y] \tag{C.26} \]

** Products of independent random variables

- When $X$ and $Y$ are independent with defined expectations:

  \begin{align*}
  \mathrm{E}[XY] &= \sum_x \sum_y xy \cdot \Pr\{X = x \text{ and } Y = y\} \\
  &= \sum_x \sum_y xy \cdot \Pr\{X = x\}\Pr\{Y = y\}
  \end{align*}

  Factor the sums:
  \[ = \left(\sum_x x \cdot \Pr\{X = x\}\right)\left(\sum_y y \cdot \Pr\{Y = y\}\right) = \mathrm{E}[X]\,\mathrm{E}[Y] \]

- Generalizes: for $n$ mutually independent random variables,
  \[ \mathrm{E}[X_1 X_2 \cdots X_n] = \mathrm{E}[X_1]\,\mathrm{E}[X_2] \cdots \mathrm{E}[X_n] \tag{C.27} \]

** Expectation for $\mathbb{N}$ valued random variables

- When $X$ takes values in $\mathbb{N} = \{0, 1, 2, \dots\}$:
  \[ \mathrm{E}[X] = \sum_{i=1}^{\infty} \Pr\{X \geq i\} \tag{C.28} \]
  Derived by rewriting $\Pr\{X = i\} = \Pr\{X \geq i\} - \Pr\{X \geq i+1\}$ and telescoping; each term $\Pr\{X \geq i\}$ is added $i$ times and subtracted $i-1$ times.

** Convexity and Jensen's inequality

- A function $f(x)$ is *convex* if for all $x, y$ and all $0 \leq \lambda \leq 1$:

  \[ f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y) \tag{C.29} \]

- *Jensen's inequality:* for a convex function $f$ applied to a random variable $X$,

  \[ \mathrm{E}[f(X)] \geq f(\mathrm{E}[X]) \tag{C.30} \]

  provided expectations exist and are finite.

** Variance and standard deviation

- Expectation does not capture spread. Example: $X$ with $\Pr\{X = 1/4\} = \Pr\{X = 3/4\} = 1/2$ and $Y$ with $\Pr\{Y = 0\} = \Pr\{Y = 1\} = 1/2$ both have $\mathrm{E}[X] = \mathrm{E}[Y] = 1/2$, but $Y$ is more spread out.

- The *variance* of $X$ with mean $\mathrm{E}[X]$:

  \[ \mathrm{Var}[X] = \mathrm{E}[(X - \mathrm{E}[X])^2] = \mathrm{E}[X^2] - \mathrm{E}^2[X] \tag{C.31} \]

  Derivation: expand $(X - \mathrm{E}[X])^2$, apply linearity, use the facts that $\mathrm{E}[\mathrm{E}^2[X]] = \mathrm{E}^2[X]$ (since $\mathrm{E}[X]$ is a constant) and $\mathrm{E}[X \cdot \mathrm{E}[X]] = \mathrm{E}^2[X]$ (by equation C.25 with $a = \mathrm{E}[X]$).

- Rearranging (C.31) gives a formula for the second moment:

  \[ \mathrm{E}[X^2] = \mathrm{Var}[X] + \mathrm{E}^2[X] \tag{C.32} \]

- Scaling: $\mathrm{Var}[aX] = a^2 \,\mathrm{Var}[X]$.

- For independent $X, Y$: $\mathrm{Var}[X + Y] = \mathrm{Var}[X] + \mathrm{Var}[Y]$.

- Generalizes: for $n$ pairwise independent random variables,

  \[ \mathrm{Var}\!\left[\sum_{i=1}^{n} X_i\right] = \sum_{i=1}^{n} \mathrm{Var}[X_i] \tag{C.33} \]

  Note: only pairwise independence is needed, not mutual independence.

- The *standard deviation* of $X$ is $\sigma_X = \sqrt{\mathrm{Var}[X]}$. The variance is then $\sigma^2$.

* C.4 The geometric and binomial distributions

** Bernoulli trials

- A *Bernoulli trial* has exactly two outcomes: *success* with probability $p$ and *failure* with probability $q = 1 - p$.

- *Bernoulli trials* (collectively): mutually independent trials, each with the same success probability $p$ (unless stated otherwise).

- Two distributions arise: the geometric distribution (trials until first success) and the binomial distribution (successes in $n$ trials).

** The geometric distribution

- Random variable $X$: number of trials needed to obtain the first success. $X \in \{1, 2, \dots\}$.

- PMF: for $k \geq 1$,

  \[ \Pr\{X = k\} = q^{k-1} p \,. \]

  Reasoning: $k - 1$ failures each with probability $q$, then one success with probability $p$.

- [FIGURE C.1: geometric distribution with $p = 1/3$, expectation $1/p = 3$]

*** Expectation of the geometric distribution

- *Expectation:*

  \[ \mathrm{E}[X] = \sum_{k=1}^{\infty} k q^{k-1} p = \frac{p}{q} \sum_{k=0}^{\infty} k q^{k} = \frac{p}{q} \cdot \frac{q}{(1-q)^2} = \frac{p}{q} \cdot \frac{q}{p^2} = \frac{1}{p} \,. \]

  Key step: uses identity $\sum_{k=0}^{\infty} k x^k = x/(1-x)^2$ for $|x| < 1$ (equation (A.11)).

- *Variance:* $\mathrm{Var}[X] = q / p^2$ (equation (C.37)).

- Example: rolling two dice until getting a 7 or 11. $p = 8/36 = 2/9$, so expected rolls $= 1/p = 9/2 = 4.5$.

** The binomial distribution

- Random variable $X$: number of successes in $n$ Bernoulli trials. $X \in \{0, 1, \dots, n\}$.

- PMF: for $k = 0, 1, \dots, n$,

  \[ \Pr\{X = k\} = \binom{n}{k} p^k q^{n-k} \,. \]

  A distribution satisfying this is a *binomial distribution*.

- Shorthand notation: $b(k; n, p) = \binom{n}{k} p^k (1-p)^{n-k}$.

- Normalization: $\sum_{k=0}^{n} b(k; n, p) = 1$, since each term is the $k$ th term in the expansion of $(p + q)^n$ and $p + q = 1$ (equation (C.4)).

- [FIGURE C.2: binomial distribution $b(k; 15, 1/3)$ with mean $np = 5$]

*** Expectation of the binomial distribution

- *Direct computation* (equation (C.41)):

  \[ \mathrm{E}[X] = \sum_{k=0}^{n} k \cdot b(k; n, p) = \sum_{k=1}^{n} k \binom{n}{k} p^k q^{n-k} \,. \]

  Use $k \binom{n}{k} = n \binom{n-1}{k-1}$ (equation (C.9)) to get:

  \[ = np \sum_{k=0}^{n-1} \binom{n-1}{k} p^k q^{(n-1)-k} = np \sum_{k=0}^{n-1} b(k; n-1, p) = np \,. \]

- *Via linearity of expectation* (simpler): let $X_i$ be the indicator for success on trial $i$. Then $\mathrm{E}[X_i] = p$ and

  \[ \mathrm{E}[X] = \sum_{i=1}^{n} \mathrm{E}[X_i] = np \,. \]

*** Variance of the binomial distribution

- Each indicator $X_i$ is Bernoulli: $X_i^2 = X_i$, so $\mathrm{E}[X_i^2] = p$.

- $\mathrm{Var}[X_i] = p - p^2 = pq$ (equation (C.43)).

- Independence of trials allows summing variances (equation (C.33)):

  \[ \mathrm{Var}[X] = \sum_{i=1}^{n} \mathrm{Var}[X_i] = npq \,. \]

*** Mode of the binomial distribution

- Ratio of successive terms (equation (C.45)):

  \[ \frac{b(k; n, p)}{b(k-1; n, p)} = \frac{(n-k+1)p}{kq} = 1 + \frac{(n+1)p - k}{kq} \,. \]

- This ratio is $> 1$ iff $(n+1)p - k > 0$, i.e. $k < (n+1)p$. So:

  - $b(k; n, p)$ increases for $k < (n+1)p$,

  - $b(k; n, p)$ decreases for $k > (n+1)p$.

- If $(n+1)p$ is an integer, two equal maxima at $k = (n+1)p$ and $k = (n+1)p - 1 = np - q$.

- Otherwise, unique maximum at the unique integer $k$ with $np - q < k < (n+1)p$.

*** Lemma C.1 — upper bound on the binomial

- *Lemma C.1:* let $n \geq 0$, $0 < p < 1$, $q = 1 - p$, and $0 \leq k \leq n$. Then

  \[ b(k; n, p) \leq \left(\frac{np}{k}\right)^k \left(\frac{nq}{n-k}\right)^{n-k} \,. \]

- *Proof:* start from the definition $b(k; n, p) = \binom{n}{k} p^k q^{n-k}$. Apply the bound $\binom{n}{k} \leq (n/k)^k \cdot (n/(n-k))^{n-k}$ (inequality (C.7)). Multiply through by $p^k q^{n-k}$ to get the result.
