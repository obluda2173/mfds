* Problem 36 :random_sampling:random_permutation:
** Task
#+begin_quote
Professor Gallup wants to create a random sample of the set $\{1, \ldots, n\}$, that is, an $m$ element subset $S$, where $1 \leq m \leq n$, such that each $m$ subset is equally likely to be created. One way is to set $A[i] = i$, for $i = 1, 2, \ldots, n$, call RANDOMLY-PERMUTE($A$), and then take just the first $m$ array elements. Show that this indeed produces a random sample.
#+end_quote

** Solution

Fix $T \subseteq \{1, \ldots, n\}$ with $|T| = m$.

RANDOMLY-PERMUTE yields each permutation with probability $1/n!$. Number of permutations whose first $m$ positions contain exactly $T$:

- arrange $T$ in positions $1, \ldots, m$: $m!$ ways,
- arrange $\{1, \ldots, n\} \setminus T$ in positions $m+1, \ldots, n$: $(n-m)!$ ways.

\[
\Pr[S = T] = \frac{m!(n-m)!}{n!} = \frac{1}{\binom{n}{m}}.
\]

Independent of $T$, so all $\binom{n}{m}$ subsets are equiprobable. $\blacksquare$

*** Numerical check

For $n = 10$, $m = 3$: $\Pr[S = T] = \frac{3!\,7!}{10!} = \frac{1}{\binom{10}{3}} = \frac{1}{120} \approx 0.00833$.

#+begin_src julia :results value
using Random
let
    n, m = 10, 3
    target = Set(1:m)               # fix an arbitrary 3-subset
    epochs = 1_000_000
    hits = 0
    for _ in 1:epochs
        sample = Set(shuffle(1:n)[1:m])
        hits += (sample == target)
    end
    hits / epochs                   # should be ≈ 1/120 ≈ 0.00833
end
#+end_src
