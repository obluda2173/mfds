* Problem 11 :DONE:
** Task
#+begin_quote
For sequences $f(n)$, $g(n)$, $h(n)$, and $k(n)$ prove that $f(n) = \Omega(h(n))$ and $g(n) = \omega(k(n))$ implies $f(n) + g(n) = \Omega(h(n) + k(n))$.
#+end_quote

** Solution

Assume $f(n) = \Omega(h(n)), \, g(n) = \omega(k(n))$. Show $f(n) + g(n) = \Omega(h(n) + k(n))$.

$$\Omega(h(n)) = \left\{ f(n) : \exists c_1, n_1 > 0 : 0 \le c_1 \cdot h(n) \le f(n), \forall n \ge n_1 \right\}$$

$$\omega(k(n)) = \left\{ g(n) : \forall c_2 > 0, \exists n_2 > 0 : 0 \le c_2 \cdot k(n) < g(n), \forall n \ge n_2 \right\}$$

Let $n_0 = \max(n_1, n_2)$ and $c_2 = c_1$. Then $\forall n \ge n_0$:

\begin{align*}
0 \le c_1 \cdot h(n) &\le f(n) \\
0 \le c_1 \cdot k(n) &< g(n) \\
0 \le c_1 \cdot \left( h(n) + k(n) \right) &\le f(n) + g(n)
\end{align*}

Hence $f(n) + g(n) = \Omega(h(n) + k(n))$. $\blacksquare$

* Problem 12 :DONE:
** Task
#+begin_quote
Let $f(n) = O(g(n))$ and $g(n) = o(h(n))$. Prove that $f(n) = o(h(n))$.
#+end_quote

** Solution

Assume $f(n) = O(g(n))$ and $g(n) = o(h(n))$. Show $f(n) = o(h(n))$.

$$O(g(n)) = \left\{ f(n) : \exists c_1, n_1 > 0 : 0 \le f(n) \le c_1 \cdot g(n), \, \forall n \ge n_1 \right\}$$

$$o(h(n)) = \left\{ g(n) : \forall c_2 > 0, \exists n_2 > 0 : 0 \le g(n) < c_2 \cdot h(n), \, \forall n \ge n_2 \right\}$$

Let $c > 0$ be arbitrary. Choose $c_2 = \frac{c}{c_1}$ and $n_0 = \max(n_1, n_2)$. Then $\forall n \ge n_0$:

\begin{align*}
0 \le f(n) &\le c_1 \cdot g(n) \\
  &< c_1 \cdot c_2 \cdot h(n) \\
  &< c_1 \cdot \frac{c}{c_1} \cdot h(n) \\
  &< c \cdot h(n)
\end{align*}

Hence $f(n) = o(h(n))$. $\blacksquare$
