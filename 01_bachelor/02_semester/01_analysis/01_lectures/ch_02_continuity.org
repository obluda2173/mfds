* Definition 11.1.11: Path-connectedness

A metric space $X$ is called path-connected if for all $x_1, x_2 \in X$ there is a continuous function $\gamma : \left[ 0,1 \right] \rightarrow X$ such that $\gamma(0) = x_1$ and $\gamma(1) = x_2$. The function $\gamma$ is called a continuous path from $x_1$ to $x_2$.

* Lemma      11.1.12: Path-connectedness is preserved under continuous maps

Let $X$ and $Y$ be metric spaces. If $X$ is path-connected and $f : X \rightarrow Y$ is continuous, then $f(X)$ is path connected.

*Proof.*

Let $a_1, a_2 \in f(X)$ be arbitrary. Then there are $x_1, x_2 \in X$ such that $f(x_1) = a_1$ and $f(x_2) = a_2$. Since $X$ is connected, there is a continuous function $\gamma : \left[ 0,1 \right] \rightarrow X$ such that $\gamma(0) = x_1$ and $\gamma(1) = x_2$. The function $f \circ \gamma : \left[ 0,1 \right] \rightarrow f(X)$ is also continuous, because $f$ is assumed to be continuous. It satisfies $f \circ \gamma(0) = f(x_1) = a_1$ and $f \circ \gamma(1) = f(x_2) = a_2$.

** Q&A

- Q_01

  Why does the definition state that $f(X)$ is path connected and not $Y$ itself?

- A_01

  The lemma states that $f(X)$ is path-connected rather than $Y$ because $f$ need not be surjective, there may be points in $Y$ that no element of $X$ maps to, so we can't say anything about connecting arbitrary points of $Y$ with paths. What the lemma captures is that continuous maps preserve path-connectedness of the image: any two points in $f(X)$ come from points in $X$, which can be joined by a path in $X$ (since $X$ is path-connected), and applying $f$ to that path gives a path in $f(X)$ joining their images.

* Theorem    11.1.13: Path-connectedness in $\mathbb{R}$ is equivalent to being an interval

Let $K \subseteq \mathbb{R}$. Then $K$ is path-connected iff it is an interval.

*Proof.*

- ($\Rightarrow$) Let $K$ be an interval. For every pair $x_1, x_2 \in K$, we may define $\gamma : \left[ 0,1 \right] \rightarrow K$ by $\gamma(t) = (1-t)x_1 + tx_2$. It is obviously continuous with $\gamma(0) = x_1$ and $\gamma(1) = x_2$, and its other values lie between $x_1$ and $x_2$, so that it does map into the interval $K$. This shows that $K$ is path-connected.

  /common sense, as we basically use the definition of the Path-connectedness to show $K$ is path-connected/

- ($\Leftarrow$) Now, suppose that $K$ is path-connected. Choose two arbitrary points $x_1, x_2 \in K$ such that $x_1 < x_2$. Let $y \in \mathbb{R}$ be an arbitrary point satisfying $x_1 < y < x_2$. We aim to show that $y$ is also in $K$.

  Since $K$ is path-connected, there is a continuous function $\gamma : \left[ 0,1 \right] \rightarrow K$ such that $\gamma(0) = x_1$ and $\gamma(1) = x_2$. We define

  $$Q = \left\{ t \in \left[ 0,1 \right] : \gamma(t) \le y \right\} \subseteq \mathbb{R}$$

  Then $Q$ is obviously bounded, hence by the order completeness of $\mathbb{R}$, the supremum exists $t_0 = sup Q$. By /Lemma 10.1.15/ there is a sequence $(q_n)_{n\in\mathbb{N}} \subseteq Q$ such that $q_n \rightarrow t_0$.

  Since $\gamma(q_n) \le y$, we also have $\gamma(t_0) \le y$. If $\gamma(t_0) < y$ were true, then we may define $\epsilon := |\gamma(t_0) - y| > 0$ and there is $\delta > 0$ such that $|\gamma(t) - \gamma(t_0)| \le \frac{\epsilon}{2}$ for all $|t - t_0| \le \delta$. This implies that $\gamma(t_0 + \delta) \le y$, which contradicts that $t_0$ is an upper bound. Hence, $\gamma(t_0) = y$.

  Since $\gamma(\left[ 0,1 \right]) \subseteq K$, we must have $y \in K$. This shows that $K$ is an interval.

* Corollary  11.1.14: Compact and path-connected subsets of $\mathbb{R}$

Let $K \subseteq \mathbb{R}$. Then $K$ is compact and path-connected if and only if it is a closed, bounded interval.

*Proof.*

This is a consequence of the /Heine-Borel Theorem 10.3.11/ and /Theorem 11.1.13/.

* Theorem    11.1.15: Intermediate value theorem

If $f : \left[ a, b \right] \rightarrow \mathbb{R}$ is continuous, then $f(\left[ a, b \right])$ is a closed, bounded interval. In particular, if $f(a) < 0$ and $f(b) > 0$, then there exists $\xi \in (a, b)$ such that

$$f(\xi) = 0.$$

*Proof.*

The interval $\left[ a, b \right]$ is compact by the /Heine-Borel Theorem 10.3.11/. /Theorem 11.1.8/ implies the image under $f$ is compact. The range $f(\left[ a, b \right])$ is path-connected by /Theorem 11.1.13/. Hence, $f(\left[ a, b \right])$ is a closed interval and we have

$$0 \in \left[ f(a), f(b) \right] \subseteq f(\left[ a, b \right]).$$

This means there is $\xi \in \left[ a, b \right]$ such that $f(\xi) = 0$.

* Theorem    11.1.19: Banach fixed point theorem

Let $(X, d)$ be a complete metric space and let $f : X \rightarrow X$ be a contraction, i.e. there exists a constant $q \in \left[ 0, 1 \right)$ such that

$$d(f(x), f(y)) \le q \, d(x, y) \quad \forall x, y \in X.$$

Then:

1. $f$ admits a unique fixed point $x \in X$, i.e. $f(x) = x$.

2. For every $x_0 \in X$, the iterates $x_{n+1} := f(x_n)$ converge linearly to the fixed point $x$.

3. One has the approximation estimates, for all $n \ge 0$,

   $$d(x_n, x) \le \frac{q^n}{1 - q} \, d(x_1, x_0).$$

*Proof.*

Fix an arbitrary $x_0 \in X$ and define the Picard iteration $x_{n+1} := f(x_n)$ for $n \ge 0$. By the contraction property,

$$d(x_{n+1}, x_n) = d(f(x_n), f(x_{n-1})) \le q \, d(x_n, x_{n-1}).$$

Iterating this inequality yields, for all $n \ge 0$,

$$d(x_{n+1}, x_n) \le q^n \, d(x_1, x_0).$$

To verify that $(x_n)_{n\in\mathbb{N}}$ is Cauchy, we consider $m > n$ and deduce

\begin{align*}
d(x_m, x_n) &\le \sum_{k=n}^{m-1} d(x_{k+1}, x_k) \\
&\le \sum_{k=n}^{m-1} q^k \, d(x_1, x_0) \\
&\le d(x_1, x_0) \sum_{k=n}^{\infty} q^k \\
&= d(x_1, x_0) \, \frac{q^n}{1 - q}.
\end{align*}

Since $q \in \left[ 0, 1 \right)$, the right-hand side tends to $0$ as $n \rightarrow \infty$, hence $(x_n)_{n\in\mathbb{N}}$ is a Cauchy sequence in $X$.

Because $X$ is complete, there exists $x \in X$ such that $x_n \rightarrow x$. To verify $f(x) = x$, we use the triangle inequality and the contraction property,

$$d(f(x), x) \le d(f(x), f(x_n)) + d(f(x_n), x) \le q \, d(x, x_n) + d(x_{n+1}, x).$$

Letting $n \rightarrow \infty$ gives $d(f(x), x) = 0$, hence $f(x) = x$.

If $y$ is another fixed point, then

$$d(x, y) = d(f(x), f(y)) \le q \, d(x, y).$$

Since $q < 1$, this implies $d(x, y) = 0$ and hence $x = y$.

The convergence is linear because

$$d(x_{n+1}, x) = d(f(x_n), f(x)) \le q \, d(x_n, x)$$

and $q \in (0, 1)$.

To derive the bound on the approximation error, we send $m \rightarrow \infty$ in our Cauchy estimates and obtain, for all $n \ge 0$,

$$d(x_n, x) \le \frac{q^n}{1 - q} \, d(x_1, x_0),$$

which in particular implies $x_n \rightarrow x$. $\square$
