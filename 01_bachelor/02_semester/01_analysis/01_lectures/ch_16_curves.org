* Definition  16.4.0: Curve

Let $X$ be a metric space. A map $f : \left[ 0,1 \right] \rightarrow X$ is also called a curve. If $X = \mathbb{R}^n$, then $f$ is called a space curve.

* Definition  16.4.1: Length of a curve in a metric space

Let $(X, d)$ be a metric space and let $\gamma : \left[ 0, 1 \right] \rightarrow X$ be a curve. For a partition

$$\mathcal{P} = \{ 0 = t_0 < t_1 < \cdots < t_n = 1 \}$$

of $\left[ 0, 1 \right]$, define the polygonal length of $\gamma$ along $\mathcal{P}$ by

$$\ell(\gamma, \mathcal{P}) := \sum_{j=1}^{n} d\left( \gamma(t_{j-1}), \gamma(t_j) \right).$$

The length of $\gamma$ is

$$\ell(\gamma) := \sup_{\mathcal{P}} \ell(\gamma, \mathcal{P}),$$

where the supremum is taken over all partitions $\mathcal{P}$ of $\left[ 0, 1 \right]$. The curve $\gamma$ is called rectifiable if $\ell(\gamma) < \infty$.

* Definition  16.4.2: Metric derivative

Let $(X, d)$ be a metric space and let $\gamma : \left[ 0, 1 \right] \rightarrow X$ be a curve. If the limit exists,

$$|\dot{\gamma}|(t) = \lim_{h \rightarrow 0} \frac{d(\gamma(t + h), \gamma(t))}{|h|},$$

then $|\dot{\gamma}|(t)$ is called the metric derivative of $\gamma$ in $t$.

* Definition  16.4.3: Lipschitz continuous curve

Let $(X, d)$ be a metric space and let $\gamma : \left[ 0, 1 \right] \rightarrow X$ be a curve. We say that $\gamma$ is Lipschitz continuous if there is a constant $L > 0$ such that

$$d(\gamma(t), \gamma(s)) \le L |t - s|, \qquad t, s \in \left[ 0, 1 \right].$$

* Theorem     16.4.4:  Arc length via metric derivative

Let $(X, d)$ be a metric space and let $\gamma : \left[ 0, 1 \right] \rightarrow X$ be a Lipschitz continuous curve. If its metric derivative exists on $\left[ 0, 1 \right]$ and is integrable, then

$$\ell(\gamma) = \int_0^1 |\dot{\gamma}|(t) \, dt$$

* Lemma       16.4.5:  Metric derivative of smooth curves

If the curve $\gamma : \left[ 0, 1 \right] \rightarrow \mathbb{R}^n$ is continuously differentiable, then it is Lipschitz continuous and its metric derivative $|\dot{\gamma}|$ satisfies

$$|\dot{\gamma}|(t) = \|\gamma'(t)\|_2$$

In particular, $|\dot{\gamma}|$ is continuous on $\left[ 0, 1 \right]$.

*Proof.*

Differentiability leads to

$$\frac{\|\gamma(t + h) - \gamma(t)\|_2}{|h|} = \left( \sum_{i=1}^{n} \frac{|\gamma_i(t + h) - \gamma_i(t)|^2}{|h|^2} \right)^{1/2} = \left( \sum_{i=1}^{n} \left| \frac{\gamma_i(t + h) - \gamma_i(t)}{h} \right|^2 \right)^{1/2} \rightarrow \left( \sum_{i=1}^{n} |\gamma_i'(t)|^2 \right)^{1/2} = \|\gamma'(t)\|_2$$

where $h \rightarrow 0$. Thus, metric derivative satisfies $|\dot{\gamma}|(t) = \|\gamma'(t)\|_2$.

To verify Lipschitz continuity, we first recall that $\left[ 0, 1 \right]$ is compact, hence $\gamma_i'(\left[ 0, 1 \right]) \subseteq \mathbb{R}$ is compact since $\gamma_i'$ is continuous. Taking the maximum over $i = 1, \ldots, n$, there is $C > 0$ such that

$$|\gamma_i'(t)| \le C, \quad t \in \left[ 0, 1 \right], \quad i = 1, \ldots, n$$

For $s, t \in \left[ 0, 1 \right]$ the mean value theorem ensures that

$$|\gamma_i(s) - \gamma_i(t)| \le C |s - t|$$

Since $d_2 \le \sqrt{n} \, d_\infty$, we deduce

$$\|\gamma(s) - \gamma(t)\|_2 \le \underbrace{\sqrt{n} \, C}_{L} |s - t|$$

$\square$

* Theorem     16.4.6:  Arc length of smooth curves

Let $\gamma : \left[ 0, 1 \right] \rightarrow \mathbb{R}^d$ be continuously differentiable. Then

$$\ell(\gamma) = \int_0^1 \sqrt{|\gamma_1'(t)|^2 + \cdots + |\gamma_n'(t)|^2} \, dt = \int_0^1 \|\gamma'(t)\|_2 \, dt$$

*Proof.*

The function $t \mapsto \|\gamma'(t)\|_2$ is continuous on $\left[ 0, 1 \right]$.

*Step 1:* $\ell(\gamma) \le \int_0^1 g(t) \, dt$. Let $\mathcal{P} = \{t_0, \ldots, t_m\}$ be a partition of $\left[ 0, 1 \right]$. For each $j$ and each component, the mean value theorem gives

$$\gamma_k(t_j) - \gamma_k(t_{j-1}) = \gamma_k'(\xi_{j,k})(t_j - t_{j-1})$$

for some $\xi_{j,k} \in (t_{j-1}, t_j)$. Hence

$$|\gamma_k(t_j) - \gamma_k(t_{j-1})| \le \sup_{t \in \left[ t_{j-1}, t_j \right]} |\gamma_k'(t)| \, (t_j - t_{j-1})$$

Therefore

$$\sqrt{\sum_{k=1}^{n} |\gamma_k(t_j) - \gamma_k(t_{j-1})|^2} \le \sup_{t \in \left[ t_{j-1}, t_j \right]} g(t) \, (t_j - t_{j-1})$$

Summing over $j$ and taking the supremum over all partitions yields

$$\ell(\gamma) \le \int_0^1 \|\gamma'(t)\|_2 \, dt$$

*Step 2:* $\ell(\gamma) \ge \int_0^1 g(t) \, dt$. Since $g$ is continuous, it is uniformly continuous. Let $\varepsilon > 0$ and choose a partition $\mathcal{P}$ with mesh sufficiently small so that

$$\sqrt{\sum_{k=1}^{n} |\gamma_k'(s) - \gamma_k'(t)|^2} < \varepsilon \quad \text{whenever } s, t \text{ lie in one subinterval}$$

For each $j$, pick $\tau_j \in \left[ t_{j-1}, t_j \right]$. Using the mean value theorem componentwise as above, we obtain

$$\left| \frac{\gamma_k(t_j) - \gamma_k(t_{j-1})}{t_j - t_{j-1}} - \gamma_k'(\tau_j) \right| \le \varepsilon \quad (k = 1, \ldots, n)$$

Hence

$$\sqrt{\sum_{k=1}^{n} \left| \frac{\gamma_k(t_j) - \gamma_k(t_{j-1})}{t_j - t_{j-1}} - \gamma_k'(\tau_j) \right|^2} \le \sqrt{n} \, \varepsilon$$

Thus

$$\sqrt{\sum_{k=1}^{n} |\gamma_k(t_j) - \gamma_k(t_{j-1})|^2} \ge (g(\tau_j) - \sqrt{n} \, \varepsilon)(t_j - t_{j-1})$$

Summing gives

$$\ell(\gamma, \mathcal{P}) \ge \sum_{j=1}^{m} g(\tau_j)(t_j - t_{j-1}) - \sqrt{n} \, \varepsilon$$

Passing to the limit (Riemann sums) and using $\ell(\gamma) \ge \ell(\gamma, P)$,

$$\ell(\gamma) \ge \int_0^1 g(t) \, dt$$

Combining both inequalities proves

$$\ell(\gamma) = \int_0^1 \sqrt{|\gamma_1'(t)|^2 + \cdots + |\gamma_n'(t)|^2} \, dt$$

$\square$

* Lemma       16.4.8:  Reparametrisation invariance of arc length

Let $f : \left[ a, b \right] \rightarrow \mathbb{R}^n$ be a continuously differentiable function and let $\phi : \left[ 0, 1 \right] \rightarrow \left[ a, b \right]$ be bijective and continuously differentiable. Then $\gamma := f \circ \phi : \left[ 0, 1 \right] \rightarrow \mathbb{R}^n$ satisfies

$$\int_a^b \sqrt{|f_1'(x)|^2 + \cdots + |f_n'(x)|^2} \, dx = \int_0^1 \sqrt{|\gamma_1'(t)|^2 + \cdots + |\gamma_n'(t)|^2} \, dt$$

*Proof.*

The function $\phi$ is either strictly increasing or decreasing. Assume it is increasing, so that $\phi'(t) \ge 0$ with $\phi(0) = a$ and $\pi(1) = b$. Substituting $x = \phi(t)$ yields

\begin{align*}
\int_0^1 \sqrt{|\gamma_1'(t)|^2 + \cdots + |\gamma_n'(t)|^2} \, dt &= \int_0^1 \sqrt{|f_1'(\phi(t)) \phi'(t)|^2 + \cdots + |f_n'(\phi(t)) \phi'(t)|^2} \, dt \\
&= \int_0^1 \sqrt{|f_1'(\phi(t))|^2 + \cdots + |f_n'(\phi(t))|^2} \, \phi'(t) \, dt \\
&= \int_a^b \sqrt{|f_1'(x)|^2 + \cdots + |f_n'(x)|^2} \, dx
\end{align*}

The case that $\phi$ is decreasing is analogous. $\square$

* Theorem     16.5.1:  Distinctive features in finite dimensions

Let $V$ be a finite-dimensional normed vector space. Then the following statements hold:

1. $V$ is complete.

2. If $\| \cdot \|_a$ and $\| \cdot \|_b$ are norms on $V$, then there exist constants $c, C > 0$ such that

   $$c \|v\|_a \le \|v\|_b \le C \|v\|_a \quad \forall v \in V$$

3. Let $W$ be another normed vector space. If $f : V \rightarrow W$ is linear, then $f$ is continuous.

*Proof.*

(ii) Linear Algebra Course.

(i) Fix a basis $B = \{b_1, \ldots, b_n\}$ of $V$ and recall the coordinate map $\chi_B : \mathbb{R}^n \rightarrow V$ given by $\chi_B(x) = \sum_{i=1}^{n} x_i b_i$. Define the coordinate norm

$$\|v\|_B := \|\chi_B^{-1}(v)\|_1, \quad v \in V$$

where $\| \cdot \|_1$ denotes the Euclidean 1-norm on $\mathbb{R}^n$.

Let $(v_k)_{k\in\mathbb{N}}$ be a Cauchy sequence in $V$. By (ii), $(v_k)_{k\in\mathbb{N}}$ is Cauchy with respect to $\| \cdot \|_B$. Setting $x_k = \chi_B^{-1}(v_k)$, we obtain a Cauchy sequence in $\mathbb{R}^n$, which converges since $\mathbb{R}^n$ is complete. Hence $v_k \rightarrow \chi_B(x) \in V$ with respect to $\| \cdot \|_B$, and therefore with respect to the original norm by (ii).

(iii) Set

$$C := \max_{1 \le i \le n} \|f b_i\|$$

For $v = \sum_{i=1}^{n} x_i b_i$ we have

$$\|f v\| \le \sum_{i=1}^{n} |x_i| \, \|f b_i\| \le C \sum_{i=1}^{n} |x_i| = C \|v\|_B$$

By (ii), $\|v\|_B \le C_1 \|v\|$ for some $C_1 > 0$, hence

$$\|f(v_n - v)\| \le C C_1 \|v_n - v\| \rightarrow 0$$

which proves continuity of $f$. $\square$

* Definition  16.6.1:  Fréchet differentiability

Let $V$ and $W$ be normed vector spaces. A function $f : V \rightarrow W$ is called Fréchet differentiable at $x \in V$ if there exists a continuous linear map $f'(x) : V \rightarrow W$ such that

$$f(x + h) = f(x) + f'(x) h + o(\|h\|), \quad h \in V$$

Since $f'(x)$ is required to be continuous, every Fréchet differentiable function is continuous.

* Theorem     16.6.5:  Chain rule

Let $V, W, W'$ be normed vector spaces. If $f : V \rightarrow W$ and $g : W \rightarrow W'$ are Fréchet differentiable, then the composition $g \circ f : V \rightarrow W'$ is Fréchet differentiable and

$$(g \circ f)'(x) = g'(f(x)) \, f'(x)$$

Here $f'(x) : V \rightarrow W$ and $g'(f(x)) : W \rightarrow W'$ are continuous linear maps, and their composition is again a continuous linear map from $V$ to $W'$.

*Proof.*

Define $\varphi(h)$ by

$$g(f(x + h)) = g(f(x)) + g'(f(x)) \, f'(x) h + \varphi(h)$$

We must show that $\varphi(h) / \|h\| \rightarrow 0$ as $h \rightarrow 0$.

Since $f$ is Fréchet differentiable at $x$, there exists a remainder $\psi(h)$ such that

$$f(x + h) = f(x) + f'(x) h + \psi(h), \quad \frac{\|\psi(h)\|}{\|h\|} \rightarrow 0 \quad (h \rightarrow 0)$$

Substituting into $g$ yields

$$\frac{\varphi(h)}{\|h\|} = \frac{g(f(x) + f'(x) h + \psi(h)) - g(f(x)) - g'(f(x)) \, f'(x) h}{\|h\|}$$

Set $\varepsilon_h := f'(x) h + \psi(h)$. Then

$$\frac{\varphi(h)}{\|h\|} = \frac{g(f(x) + \varepsilon_h) - g(f(x)) - g'(f(x)) \, \varepsilon_h}{\|h\|} + \frac{g'(f(x)) \, \psi(h)}{\|h\|}$$

The second term tends to zero since $g'(f(x))$ is continuous and $\psi(h) / \|h\| \rightarrow 0$.

For the first term we write

$$\frac{g(f(x) + \varepsilon_h) - g(f(x)) - g'(f(x)) \, \varepsilon_h}{\|h\|} = \frac{\|\varepsilon_h\|}{\|h\|} \cdot \frac{g(f(x) + \varepsilon_h) - g(f(x)) - g'(f(x)) \, \varepsilon_h}{\|\varepsilon_h\|}$$

Since $f'(x)$ is continuous, $\varepsilon_h \rightarrow 0$ as $h \rightarrow 0$, and the second factor tends to zero by the Fréchet differentiability of $g$. Moreover,

$$\frac{\|\varepsilon_h\|}{\|h\|} \le \|f'(x)\| + \frac{\|\psi(h)\|}{\|h\|}$$

which is bounded for small $h$. Hence $\varphi(h) / \|h\| \rightarrow 0$, proving the claim. $\square$
