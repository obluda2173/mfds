* Problem 1 :DONE:
** Task
Verify the following trigonometric identities and check them in Julia.
$\cos(-x)=\cos(x)$
$\sin(-x)=-\sin(x)$
$\cos(x)=\sin(x+\frac{\pi}{2})$
$\sin(x)=\cos(x-\frac{\pi}{2})$
$\sin(x+\pi)=-\sin(x)$
$\cos(x+\pi)=-\cos(x)$,

** Notes

** Solution
*** $\cos(-x)=\cos(x)$
#+begin_quote
Tailor Series Expansion
$$
cos(x) = \sum_{k=0}^{\infty} \dfrac{(-1)^k x^{2k}}{(2k)!} \qquad
sin(x) = \sum_{k=0}^{\infty} \dfrac{(-1)^k x^{2k+1}}{(2k+1)!} \qquad
$$
#+end_quote

Lets assume $cos(-x) = cos(x)$. Hence,

$\sum_{k=0}^{\infty} \dfrac{(-1)^k (-x)^{2k}}{(2k)!} = \sum_{k=0}^{\infty} \dfrac{(-1)^k x^{2k}}{(2k)!}$

Since $k \in \mathbb{N}_0$, $(-x)^{2k} = x^{2k}$. Thus,

$\sum_{k=0}^{\infty} \dfrac{(-1)^k x^{2k}}{(2k)!} = \sum_{k=0}^{\infty} \dfrac{(-1)^k x^{2k}}{(2k)!}$

$cos(-x) = cos(x)$

*** $\sin(-x)=-\sin(x)$
#+begin_quote
Tailor Series Expansion
$$
cos(x) = \sum_{k=0}^{\infty} \dfrac{(-1)^k x^{2k}}{(2k)!} \qquad
sin(x) = \sum_{k=0}^{\infty} \dfrac{(-1)^k x^{2k+1}}{(2k+1)!} \qquad
$$
#+end_quote

$sin(-x) = \sum_{k=0}^{\infty} \dfrac{(-1)^k (-x)^{2k+1}}{(2k+1)!}$

Expression $2k+1$, where $k \in \mathbb{N}_0$ would always be odd.

$sin(-x) = -\sum_{k=0}^{\infty} \dfrac{(-1)^k (x)^{2k+1}}{(2k+1)!}$

$sin(-x) = -sin(x)$

*** $\cos(x)=\sin(x+\frac{\pi}{2})$
#+begin_quote
Trigonometric identity:
$$\sin(A + B) = \sin(A)\cos(B) + \cos(A)\sin(B)$$
#+end_quote

$sin(x + \dfrac{\pi}{2}) = sin(x)cos(\dfrac{\pi}{2}) + cos(x)sin(\dfrac{\pi}{2})$

$sin(x + \dfrac{\pi}{2}) = cos(x)$

*** $\sin(x)=\cos(x-\frac{\pi}{2})$
#+begin_quote
Trigonometric identity:
$$\cos(A - B) = cos(A)cos(B) + sin(A)sin(B)$$
#+end_quote

$\cos(x - \dfrac{\pi}{2}) = cos(x)cos(\dfrac{\pi}{2}) + sin(x)sin(\dfrac{\pi}{2})$

$\cos(x - \dfrac{\pi}{2}) = sin(x)$

*** $\sin(x+\pi)=-\sin(x)$
#+begin_quote
Trigonometric identity:
$$\sin(A + B) = \sin(A)\cos(B) + \cos(A)\sin(B)$$
#+end_quote

$\sin(x + \pi) = \sin(x)\cos(\pi) + \cos(x)\sin(\pi)$

$\sin(x + \pi) = -\sin(x)$

*** $\cos(x+\pi)=-\cos(x)$
#+begin_quote
Trigonometric identity:
$$\cos(A + B) = cos(A)cos(B) - sin(A)sin(B)$$
#+end_quote

$\cos(x + \pi) = cos(x)cos(\pi) - sin(x)sin(\pi)$

$\cos(x + \pi) = -cos(x)$

using Symbolics

euler_cos(θ) = (exp(im*θ) + exp(-im*θ)) / 2
euler_sin(θ) = (exp(im*θ) - exp(-im*θ)) / (2im)

function validate(label, LHS, RHS)
    # symbolic difference
    diff = LHS - RHS

    sym_diff = simplify(diff)

    @variables x
    num_diff = substitute(sym_diff, Dict(x => 1.0))

    val = Symbolics.unwrap(num_diff)

    if abs(val) < 1e-12
        println(rpad(label, 30), " ✅ Q.E.D.")
    else
        println(rpad(label, 30), " ❌ Not proven. Diff ≈ $val")
    end
end

@variables x

println("--- PROOFS (Euler + Numerical Check) ---")
validate("cos(-x) = cos(x)",       euler_cos(-x), euler_cos(x))
validate("sin(-x) = -sin(x)",      euler_sin(-x), -euler_sin(x))

* Problem 5 :DONE:
** Task
Prove by induction (and using the product rule) that, for $1\le n\in\mathbb{N}$ the derivative of $f:\mathbb{R}\rightarrow\mathbb{R}$, $f(x)=x^{n}$ is given by $f^{\prime}(x)=nx^{n-1}$.

** Notes
#+begin_quote
*Proposition* The statement S1, S2, S3, S4, ... are all true.

Proof. (Induction)
1. Prove that the first statement S1 is true
2. Given any integer k >= 1, prove that the statement Sk => Sk+1 is true.

It follows by mathematical induction that every Sn is true.
#+end_quote

** Solution
*Proposition.* For $1 \le n \in \mathbb{N} : f : \mathbb{R} \rightarrow \mathbb{R}, f(x) = x^n \implies f'(x) = nx^{n-1}$.

*Proof.*

*Base Case.* $n = 1$.
$f(x) = x^1 = x$.
Using the limit definition:
$f'(x) = \lim_{h\rightarrow0} \dfrac{(x+h) - x}{h} = \lim_{h\rightarrow0} \dfrac{h}{h} = 1$.

Using the formula:
$nx^{n-1} = 1x^{1-1} = 1$.

Base holds.

*Induction Hypothesis.* Assume statement holds for $n = k : \dfrac{d}{dx}(x^k) = kx^{k-1}$.

*Inductive Step.*
We must show that the statement holds for $n = k+1$.
Target: $\dfrac{d}{dx}(x^{k+1}) = (k+1)x^k$.

Rewrite the term:
$x^{k+1} = x \cdot x^k$

Apply the Product Rule ($u=x, v=x^k$):
$\dfrac{d}{dx}(x^{k+1}) = \dfrac{d}{dx} \left[ x \cdot x^k \right]$

$\dfrac{d}{dx}(x \cdot x^k) = \left[ \dfrac{d}{dx} x \right] \cdot x^k + x \cdot \left[ \dfrac{d}{dx} x^k \right]$

Substitute derivatives (Base Case and Hypothesis):
$\dfrac{d}{dx}(x \cdot x^k) = (1) \cdot x^k + x \cdot (kx^{k-1})$

Simplify:
$\dfrac{d}{dx}(x^{k+1}) = x^k + kx^{1 + k - 1}$
$\dfrac{d}{dx}(x^{k+1}) = x^k + kx^k$

Factor:
$\dfrac{d}{dx}(x^{k+1}) = (1+k)x^k$
$\dfrac{d}{dx}(x^{k+1}) = (k+1)x^k$

*Conclusion.* Since $S_1$ is true and $S_k \implies S_{k+1}$, the proposition is true for all $n \in \mathbb{N}$ by mathematical induction.

* Problem 6 :DONE:
** Task
Compute the derivative of the following functions:
1. $f(x)=\sin(x^{2})$
2. $g(x)=e^{3x+1}$
3. $f(x)=\ln(5x^{2}-x+4)$
4. $g(x)=(2x^{3}-x)^{5}$
5. $f(x)=\sqrt{1+e^{2x}}$
6. $g(x)=\cos(\ln~x)$, $x>0$
Visualize all functions and their derivatives in Julia.

** Notes
#+begin_quote
Chain Rule:

$$\dfrac{d}{dx} \left[ f(g(x)) \right] = f'(g(x)) \cdot g'(x)$$

Product Rule:

$$\dfrac{d}{dx} \left[ f(x)g(x) \right] = f(x)g'(x) + f'(x)g(x)$$

Derivatives of Logarithmic Functions:

$$\dfrac{d}{dx} log_a \left[ f(x) \right] = \dfrac{1}{f(x) ln(a)} \cdot f'(x)$$


#+end_quote

** Solution
*** $f(x)=\sin(x^{2})$

Using the chain rule:
$f'(x) = \cos(x^2) \cdot \frac{d}{dx}(x^2)$
$f'(x) = 2x \cos(x^2)$

*** $g(x)=e^{3x+1}$

Using the chain rule:
$g'(x) = e^{3x+1} \cdot \frac{d}{dx}(3x+1)$
$g'(x) = 3e^{3x+1}$

*** $f(x)=\ln(5x^{2}-x+4)$

Using the chain rule ($\frac{u'}{u}$):
$f'(x) = \frac{\frac{d}{dx}(5x^2 - x + 4)}{5x^2 - x + 4}$
$f'(x) = \frac{10x - 1}{5x^2 - x + 4}$

*** $g(x)=(2x^{3}-x)^{5}$

Using the general power rule:
$g'(x) = 5(2x^3 - x)^4 \cdot \frac{d}{dx}(2x^3 - x)$
$g'(x) = 5(2x^3 - x)^4 (6x^2 - 1)$

*** $f(x)=\sqrt{1+e^{2x}}$

Rewrite as $f(x) = (1+e^{2x})^{\frac{1}{2}}$.
$f'(x) = \frac{1}{2} (1 + e^{2x})^{-\frac{1}{2}} \cdot \frac{d}{dx}(1+e^{2x})$
$f'(x) = \frac{1}{2\sqrt{1 + e^{2x}}} \cdot 2e^{2x}$
$f'(x) = \frac{e^{2x}}{\sqrt{1 + e^{2x}}}$

*** $g(x)=\cos(\ln x)$, $x>0$

Using the chain rule:
$g'(x) = -\sin(\ln x) \cdot \frac{d}{dx}(\ln x)$
$g'(x) = -\frac{\sin(\ln x)}{x}$

* Problem 9 :DONE:
** Task
Consider $f:\mathbb{R}\rightarrow\mathbb{R}$ $f(x)=\cos(x)$ and the sequence
$x_{0}=2$, $x_{n+1}=x_{n}-\frac{f(x_{n})}{f^{\prime}(x_{n})}$
$n\in\mathbb{N}$
Implement the sequence in Julia and compare with the bisection method in the interval [0, 2].

** Solution
#+begin_src julia :results output :export both :tangle julia/p_9.jl
using Plots

# ensure output directory exists
mkpath("./figs")

# define functions
f(x) = cos(x)
df(x) = -sin(x)
newton_step(x) = x - f(x) / df(x)

# shared settings
iterations = 5

# --- part 1: Newton's Method ---
x0 = 2.0
n_history = Float64[x0]

for i in 1:iterations
    current_x = last(n_history)
    next_x = newton_step(current_x)

    push!(n_history, next_x)
    println("Newton Iter $i: $next_x")
end

# --- part 2: Bisection Method ---
a, b = 0.0, 2.0
b_history = Float64[]

curr_a, curr_b = a, b

for i in 1:iterations
    global curr_a, curr_b

    c = (curr_a + curr_b) / 2
    push!(b_history, c)

    if f(c) * f(curr_a) < 0
        curr_b = c
    else
        curr_a = c
    end
    println("Bisection Iter $i: $c" )
end


# plotting
indices_n = 0:(length(n_history) - 1)
indices_b = 1:length(b_history)

plot(title="Convergence Comparison: Newton vs Bisection",
     xlabel="Iteration", ylabel="Value", legend=:topright)

# plot Newton
plot!(indices_n, n_history,
      label="Newton (x_n)",
      linewidth=2, marker=:circle, color=:blue)

# plot Bisection
plot!(indices_b, b_history,
      label="Bisection (midpoints)",
      linewidth=2, marker=:square, color=:green, linestyle=:dash)

# reference Line
hline!([pi/2], label="True Root (π/2)", color=:red, linestyle=:dot)

savefig("./figs/p_9")
