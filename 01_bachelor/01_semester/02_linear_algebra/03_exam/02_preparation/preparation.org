function forward_substitution(L, b)
    n = size(L, 1)
    y = zeros(Float64, n)

    for i in 1:n
        sum = b[i]

        for j in 1:(i-1)
            sum -= L[i, j] * y[j]
        end

        y[i] = sum / L[i, i]
    end
    return y
end

function backward_substitution(U, y)
    n = size(U, 1)
    x = zeros(Float64, n)

    for i in n:-1:1
        sum = y[i]

        for j in (i+1):n
            sum -= U[i, j] * x[j]
        end
        x[i] = sum / U[i, i]
    end
    return x
end

L = [ 1  0  0;

     -1  2  1]

U = [ 2  1 -1;
      0  3 -1;
      0  0 -1]

      7;
      3]

y = forward_substitution(L, b)
x = backward_substitution(U, y)

println("Intermediate vector y: ", y)
println("Final solution x: ", x)

*** Solution

Let $V$ be a vector space over a field $\mathbb{F}$. Let $L = \left\{ v_1, v_2, \ldots, v_m \right\}$ be linearly independent and $S = \left\{ w_1, w_2, \ldots, w_{n} \right\}$ be a spanning set over $V$. Then $m \le n$, and there exists $S' \subseteq S$ with $|S'| = n - m$ such that $S' \cup L$ spans $V$.

- *Custom Exam Style Proof.*

  - *Theorem.*

    Let $L = \left\{ v_1, \ldots, v_m \right\}$ be linearly independent and $S = \left\{ w_1, \ldots, w_n \right\}$ span $V$. Then $m \leq n$, and $\exists S' \subseteq S$ with $|S'| = n - m$ such that $L \cup S'$ spans $V$.

  - *Proof.*

    $m = 0$: Trivial, $L \cup S = S$ spans $V$ and $0 \leq n$.

    Assume case holds for $m-1$. Applied to $\left\{ v_1, \ldots, v_{m-1} \right\}$ and $S$, we have obtain $S_{m-1} \subseteq S$ with $|S_{m-1}| = n - (m - 1)$ such that $\left\{ v_1, \ldots, v_{m-1} \right\} \cup S_{m-1}$ spans $V$.

*Formal Proof.*

- *Lemma (Exchange Lemma).*

  Let $V$ be a $\mathbb{F}$ - vector space, $A \subseteq V$ a subset, and $v \in \langle A \rangle$ an arbitrary element.

  ~here we just defined that $V$ is our vector space, $A$ is a subset of $V$ (because of notation $\langle A \rangle$), and let $v$ be some vector that lies in the span of $A$~

  Let $a_0 \in A$ be an element such that there is a representation of $v$ as a linear combination of elements of $A$ in which $a_0$ has a coefficient $\lambda_0 \neq 0$.

  ~here we are basically linking $a_0$ to arbitrary element $v$, by saying that $v$ can be represented as a linear combination, where $a_0$ takes place (as $\lambda_0 \neq 0$)~

  Then the set $A' := (A \setminus \left\{ a_0 \right\}) \cup \left\{ v \right\}$, which is obtained by replacing the vector $a_0$ in $A$ by $v$, satisfies $\langle A' \rangle = \langle A \rangle$.

  ~here we are constructing a new set $A'$ which is a set $A$ with replaced $v$ over $a_0$, then saying that $\langle A' \rangle = \langle A \rangle$ (meaning they have a same span, which implies $A'$ being a spanning set over $V$)~

- *Lemma Proof (Exchange Lemma).*

  By assumption we have $v \in \langle A \rangle$, hence $A' \subseteq \langle A \rangle$, hence $\langle A' \rangle \subseteq \langle A \rangle$.

  ~here as I can see we are starting to prove from both sides (e.g. $\langle A' \rangle \subseteq \langle A \rangle$ and $\langle A \rangle \subseteq \langle A' \rangle$ implies $\langle A \rangle = \langle A' \rangle$). since $v$ was assumed to be a vector that lies in the span of $A$, $A'$ must be a subset of span $A$~

  Conversely, by assumption there exists elements $a_1, \ldots, a_n \in A \setminus \left\{ a_0 \right\}$ and scalars $\lambda_0, \ldots, \lambda_n$ with $\lambda_0 \neq 0$ such that $v = \lambda_0 a_0 + \sum_{j=1}^{n} \lambda_j a_j$.

  ~here we are just saying that since by assumption $v$ is a linear combination of span $A$, we can write $v$ in the form of $v = \lambda_0 a_0 + \sum_{j=1}^{n} \lambda_j a_j$~

  Hence, $a_0 = \lambda_0^{-1} \left( v - \sum_{j=1}^n \lambda_j a_j \right) \in \langle A' \rangle$.

  ~here we have just rewrote the previous equiation for $a_0$, since the RHS is fully constructed of $v$ and $a_j$, remembering that $A' := (A \setminus \left\{ a_0 \right\}) \cup \left\{ v \right\}$ we conclude that $a_0 \in \langle A' \rangle$~

  Since obviously $A \setminus \left\{ a_0 \right\} \subseteq \langle A' \rangle$ holds, we obtain $A \subseteq \langle A' \rangle$ and therefore $\langle A \rangle \subseteq \langle A' \rangle$.

  ~here we stated obvious statement, and by the work done before (that $a_0 \in \langle A' \rangle$) stated $\langle A \rangle \subseteq \langle A' \rangle$, hence the two sided proof finishes~

- *Theorem (Steinitz Exchange Theorem).*

  Let $V$ be a finitely generated $\mathbb{F}$ vector space, and let $\left\{ v_1, \ldots, v_n \right\}$ be a spanning set of $V$. Then:

  a. If $A \subseteq V$ is a linearly independent subset, then $A$ is finite and has at most $n$ elements. If $A = \left\{ a_1, \ldots, a_k \right\}$ with $k < n$, then one can renumber the elements $v_i$, in such a way $\left\{ a_1, \ldots, a_k, v_{k+1}, \ldots v_n \right\}$ is a spanning set of $V$.

  b. If $A \subseteq V$ is an arbitrary spanning set, then there exists a finite subset $A' \subseteq A$ with at most $n$ elements that is also a spanning set of $V$.

*** Solution
**** (a)

Let $T : V \rightarrow W$ be a linear map between vector spaces $V$ and $W$ over a field $\mathbb{F}$.

- *Kernel of $T$:*

  $$ker(T) = \left\{ v \in V : T(v) = 0_{W} \right\}$$

  A kernel is the preimage of a zero vector in $W$. It is a subspace of $V$.

- *Image of $T$:*

  $im(T) = \left\{ T(v) : v \in V \right\} = T(V)$

  The image is the set of all outputs of $T$. It is a subspace of $W$.

- *Rank of $T$:*

  $$rank(T) = dim(im(T))$$

  Rank is the dimension of the image space.

**** Annotated Rewriting

*Theorem (Dimension Theorem for Linear Maps).*

Let $V$ and $W$ be finite-dimensional $\mathbb{F}$ vector spaces, and let $f : V \rightarrow W$ be linear. Then

$$dim(V) = dim(im(f)) + dim(ker(f))$$

*Proof.*

We choose a basis $\left\{ b_1, \ldots, b_k \right\}$ for the subspace $ker(f) \subseteq V$.

~ok, setting up basis for kernel~

Since the vectors $b_i$ are linearly independent, /Corollary III.1.48(ii)/ shows they can be extended to a basis $\left\{ b_1, \ldots, b_n \right\}$ of the vector space $V$.

~ok, so we now state (common sense) that the basis of kernel can be extended to span the whole vector space (keeping in mind all vectors are linearly independent)~

We claim that the vectors $f(b_{k+1}), \ldots, f(b_{n}) \in W$ form a basis of the subspace $im(f) \in W$, which of course proves the theorem.

~here we say that those extended basis vectors after the linear mapping all together form a basis for a subspace $im(f) \subseteq W$, which is logical~

First, /Proposition III.2.11/ tells us that vectors $f(b_1), \ldots, f(b_n)$ form a generating system of $im(f)$.

~so a basis of vector space after linear mapping spans the image, got it~

By construction, however, we have $f(b_1) = \ldots = f(b_k) = 0$, so $f(b_{k+1}), \ldots, f(b_n)$ also form a generating system.

~mentioning that after linear mapping, some of vectors (that form kernel) result it a 0, so only $b_{k+1}, ...$ are actually spanning the image~

On the other hand, assume that $\lambda_{k+1}, \ldots, \lambda_n \in \mathbb{F}$ are scalars such that $\lambda_{k+1}f(b_{k+1}) + \ldots + \lambda_{n}f(b_n) = 0$ holds.

~just assuming there are scalars that scale the "contributing" vectors $f(b_i)$ that sum tehm all up to 0~

Then we can write the left hand side as $f(\lambda_{k+1}b_{k+1} + \ldots + \lambda_n b_n)$, hence, $\lambda_{k+1}b_{k+1} + \ldots + \lambda_n b_n \in ker(f)$.

~now we perform an operation to put all those scalars and vectors indside of linear map, which of course results in a 0, which implies that this vector sum is an element of a kernel~

By construction, the vectors $b_1, \ldots, b_k$ form a basis of this subspace, so we find scalars $\mu_1, \ldots, \mu_k \in \mathbb{F}$ such that

$$\lambda_{k+1}b_{k+1} + \ldots + \lambda_nb_n = \mu_1b_1 + \ldots + \mu_kb_k$$

~since the vectors $b_1, \ldots, b_k$ are a basis of a kernel, there exists scalars that can result in linear combination equal to previously found one (made of $b_{k+1}, \ldots$ and scalars)~

But this means precisely that

$$\mu_1b_1 + \ldots + \mu_kb_k - \lambda_{k+1}b_{k+1} - \ldots - \lambda_nb_n = 0$$

~just rewriting previous formula~

and by linear independence of $b_1, \ldots, b_n$ this is possible only if

$$\mu_1 = \ldots = \mu_k = \lambda_{k+1} = \ldots = \lambda_n = 0$$

~and now stating that it is only possible if all of the scalars are equal to zero, meaning that vectors $f(b_{k+1}), \ldots, f(b_{n})$ are linearly independent, as their sum can be equal to 0 only if their scalars are equal to 0~

*** Solution

**** Annotated Rewriting

Let $V$ and $W$ be vector spaces over $\mathbb{F}$ and let $f : V \rightarrow W$ be linear. Then,

a. If $\hat{V} \subseteq V$ is a subspace, then $f(\hat{V}) \subseteq W$ is also a subspace. In particular $im(f) \subseteq W$ is a subspace.
b. If $\hat{W} \subseteq W$ is a subspace, then $f^{-1}(\hat{W}) \subseteq V$ is also a subspace. In particular, $ker(f) \subseteq V$ is a subspace.
c. $f$ is surjective if and only if $im(f) = W$.
d. $f$ is injective if and only if $ker(f) = \left\{ 0 \right\}$.

*Proofs.*

a. Let $w, w' \in f(\hat{V})$ and $\lambda \in \mathbb{F}$.

   /define $w$, $w'$ to be elements of $f(\hat{V})$ (or saying differently, image of subspace $\hat{V}$ under $f$) and $\lambda$ (which is element/scalar of a field) to be used for subspace criterion (where given any linear combination $\lambda w + w' \in f(\hat{V})$, given $w, w' \in f(\hat{V}), \, \lambda \in \mathbb{F}$)/

   Then there exist $v, v' \in \hat{V}$ with $f(v) = w$ and $f(v') = w'$.

   /letting $w, w' \in f(\hat{V})$ strictly implies existence of $v, v' \in \hat{V} : f(v) = w, f(v') = w'$/

   By linearity of $f$ we obtain $\lambda w + w' = \lambda f(v) + f(v') = f(\lambda v + v')$.

   /linearity of $f$ (linear map) implies $\lambda f(v) + f(v') = f(\lambda v + v')$ (remember $\lambda w + w' = \lambda f(v) + f(v')$)/

   Since $\hat{V}$ is a subspace, we have $\lambda v + v' \in \hat{V}$, and hence $\lambda w + w' \in f(\hat{V})$.

   /since $\hat{V}$ is a subspace (comes from "if"), $\lambda v + v' \in \hat{V}$ and thus, $\lambda w + w' \in f(\hat{V})$/

   Since also $f(0) = 0 \in f(\hat{V})$, it follows that $f(\hat{V})$ is a subspace.

   /as $\hat{V}$ is a subspace $0 \in \hat{V}$ and $f(0) = 0$ (linearity of $f$) hence $0 \in f(\hat{V})$/

b. Let $v, v' \in f^{-1}(\hat{W})$ and $\lambda \in \mathbb{F}$.

   /defining tools for subspace criterion, in this case for $f^{-1}(\hat{W})$ (preimage of $\hat{W}$ under $f^{-1}$)/

   Then $f(v), f(v') \in \hat{W}$, and therefore, $f(\lambda v + v') = \lambda f(v) + f(v') \in \hat{W}$,

   /letting $v, v' \in f^{-1}(\hat{W})$ strictly implies $f(v), f(v') \in \hat{W}$. linearity of $f$ lets $f(\lambda v + v') = \lambda f(v) + f(v')$ (which are elements of $\hat{W}$). since $\hat{W}$ is a subspace (by "if") it contains any linear combination $\lambda f(v) + f(v')$/

   since $\hat{W}$ is a subspace, and thus $\lambda v + v' \in f^{-1}(\hat{W})$.

   /since $\hat{W}$ is a subspace, $f(\lambda v + v') \in \hat{W}$ implies there must exist an element $(\lambda v + v') \in f^{-1}(\hat{W})$ (in a preimage)/

   We have $0 \in f^{-1}(\hat{W})$ because $f(0) = 0 \in \hat{W}$.

   /as $f(0) = 0 \in \hat{W}$, following the same logic, $0 \in f^{-1}(\hat{W})$, covering second part of subspace criterion/

   Therefore $f^{-1}(\hat{W})$ is a subspace.

c. Is a definition of surjectiveness.

d. If $f$ is injective, then the preimage of every singleton set has at most one element, so since $f(0) = 0$ it must be that $f^{-1}(\left\{ 0 \right\}) = \left\{ 0 \right\}$.

   /proof will be two sided, as proving if and only if/

   /first part of proof, pretty straight forward, if $f$ is injective, then every preimage element is "linked" to unique image element, hence, as $f(0) = 0$, it must be that the preimage of $0$ (im image) is $\left\{ 0 \right\}$ only/

   Now assume that $ker(f) = \left\{ 0 \right\}$.

   /second part of proof, assume kernel of $f$ is $\left\{ 0 \right\}$ only/

   Let $x, y \in V$ with $f(x) = f(y)$.

   /letting $x, y \in V$ with $f(x) = f(y)$ (because we will need to show $x = y$, key idea of injectiveness, no elements of preimage are mapped to the same element of image)/

   Then we obtain $0 = f(x) - f(y) = f(x - y)$, and hence $x - y \in ker(f)$, therefore $x - y = 0$, so $x = y$.

   /now we define $0$ in image through the difference of $f(x)$ and $f(y)$, and due to linearity of $f$, $f(x) - f(y) = f(x-y) = 0$, hence, $x-y$ must be element of $ker(f)$, and since kernel consist of only $\left\{ 0 \right\}$, $x$ must be equal to $y$/

   Thus $f$ is injective.
