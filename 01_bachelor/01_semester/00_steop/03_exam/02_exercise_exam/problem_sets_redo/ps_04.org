* Problem 1 [DONE]
** Task
Let x and y be nonnegative numbers. Prove that if x <= y, then sqrt{x} <= sqrt{y}.

** Solution
#+begin_quote
*Proposition.* x,y >= 0, if x <= y, then sqrt{x} <= sqrt{y}.

*Proof.* Assume x <= y with x,y >= 0.

Square both sides of sqrt{x} <= sqrt{y}:

sqrt{x}^2 = x
sqrt{y}^2 = y

Squaring is allowed since both sides are non-negative.

Thus, from x <= y, we get:

sqrt{x}^2 <= sqrt{y}^2

Since the square function is increasing on [0, ∞), this is equivalent to:

sqrt{x} <= sqrt{y}
#+end_quote

* Problem 2 [DONE]
** Task
For n ∊ ℕ prove that 4|(1 + (-1)^n * (2n - 1)).

** Solution
#+begin_quote
*Proposition.* For n ∈ ℕ, 4|(1 + (-1)^n * (2n - 1)) <--> (1 + (-1)^n * (2n - 1)) ≡ 0 (mod 4).

*Proof.*

(1 + (-1)^n * (2n - 1))

n = 1 : (1 + (-1)^1 * (2*1 - 1)) = (1 - 1(2 - 1)) = 0
n = 2 : (1 + (-1)^2 * (2*2 - 1)) = (1 + 1(4 - 1)) = 4
n = 3 : (1 + (-1)^3 * (2*3 - 1)) = (1 - 1(6 - 1)) = -4
n = 4 : (1 + (-1)^4 * (2*4 - 1)) = (1 + 1(8 - 1)) = 8
n = 5 : (1 + (-1)^5 * (2*5 - 1)) = (1 - 1(10 - 1)) = -8

Let n = 2k for k ∈ ℕ.
        (1 + (-1)^(2k) * (2*2k - 1)) = (1 + 1 * (4k - 1))
                                     = 1 + 4k - 1
                                     = 4k

Let n = 2k + 1 for k ∈ ℕ.
        (1 + (-1)^(2k+1) * (2*(2k+1) - 1)) = (1 - 1 * (4k + 2 - 1))
                                           = 1 - 4k - 1
                                           = -4k

*Conclusion.* In both cases, expression is a multiple of 4 for every n ∈ ℕ.
#+end_quote

* Problem 3 [DONE]
** Task
Prove that log2(3) is irrational.

** Solution
#+begin_quote
*Proposition.* log2(3) != a/b, a ∈ ℤ, b ∈ ℕ. log2(3) is irrational.

*Proof.*
Assume log2(3) is rational.

log2(3) = a/b, a ∈ ℤ, b ∈ ℕ.

Thus,
2^(a/b) = 3
2^a = 3^b

LHS has only one prime number factor 2
RHS has only one prime number factor 3

By the Fundamental Theorem of Arithmetics (unique prime factorisation), the two sides can not be equal unless both are 1.

But 2^p = 1 would force a = 0, which gives 3^b = 1, and then b = 0, which is impossible because of b being natural.
#+end_quote

* Problem 4 [DONE]
** Task
Five children found nine mushrooms. Prove that at least two of them found an equal number of mushrooms.

** Solution
#+begin_quote
*Proposition.* 9 items can not be splitted into 5 groups of different count number.

Let 1st brat have 0 mushrooms, this would force 2nd brat to have 0+1 mushrooms, so it would not be equal to the first brat's mushrooms. Same logic goes for the rest brats.

        1st = 0 mushrooms
        2nd = 1 mushrooms
        3rd = 2 mushrooms
        4th = 3 mushrooms
        5th = 4 mushrooms

Thus, the total amount of mushrooms in this case would be:

        0 + 1 + 2 + 3 + 4 = 10

Thus, it would force us to decrement on of the brat's mushrooms by 1. Which would lead to 2 brats with the same amount of mushrooms.
#+end_quote

* Problem 6 [DONE]

Find digits a and b such that (0.aa..)^(1/2) = 0.bb..

** Solution
0.aa.. = a/9
0.bb.. = b/9

Thus,
sqrt{a/9} = b/9
sqrt{a} * sqrt{1/9} = b/9
sqrt{a} * 1/3 = b/9             (multiply by 9)
sqrt{a} * 3 = b
3 * sqrt{a} = b

Since b is an integer, and 0 <= b <= 9, sqrt{a} should also be an integer.

Hence, a ∈ {0, 1, 4, 9}

Thus,
a = 0, 3 * sqrt{0} = 0, b = 0
a = 1, 3 * sqrt{1} = 3, b = 3
a = 4, 3 * sqrt{4} = 6, b = 6
a = 9, 3 * sqrt{9} = 9, b = 9

a. gcd(10223; 33341)

a. gcd(10223; 33341)
   33341 = 10223 * 3 + 2672
   10223 = 2672 * 3 + 2207
   2672 = 2207 * 1 + 465
   2207 = 465 * 4 + 347
   465 = 347 * 1 + 118

   118 = 111 * 1 + 7
   111 = 7 * 15 + 6
   7 = 6 * 1 + 1
   5 = 1 * 5

   Thus, 1.

b. gcd(2n + 1; 3n + 2)

   3n + 2 - 2n - 1 = n + 1

   Thus,
   gcd(2n + 1; 3n + 2) = gcd(2n + 1, n + 1)

   Now,
   2n + 1 - 2(n + 1) = - 1

   Hence,
   gcd(2n + 1, n + 1) = gcd(n + 1, 1)

   Finally,
   gcd(n + 1, 1) = 1

   Hence,
   gcd(2n + 1; 3n + 2) = 1
