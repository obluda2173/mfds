* Problem 2 [DONE] [ON BOARD]
** Task
A function f : ℤ x ℤ -> ℤ is defined as f(m,n) = 3n - 4m. Verify whether this function is injective and whether it is surjective.

** Solution
- *Injectivity*
  Assume ∃a,b ∈ ℤ^2 : f(a) = f(b). Thus,

  f(a1, a2) = 3(a1) - 4(a2)
  f(b1, b2) = 3(b1) - 4(b2)

  3(a1) - 4(a2) = 3(b1) - 4(b2)
  3(a1 - b1) - 4(a2 - b2) = 0

  This leads to:
  - a1 = b1 and a2 = b2

  - (a1 - b1) = 4/3 and (a2 - b2) = 3/4
