** Section A (1 point each)
1. **(b)** $\aleph_0$. The set of functions $\{0,1\} \to \mathbb{N}$ is equivalent to $\mathbb{N} \times \mathbb{N}$, which is countable. (Note: Functions $\mathbb{N} \to \{0,1\}$ would be uncountable).
2. **(b)** $\exists x \forall y (x + y \neq 0)$. Negation of $\forall \exists$ is $\exists \forall \neg$.
3. **(a)** $-\pi/2$. $(1-i) = \sqrt{2}e^{-i\pi/4}$. Raised to 10: $2^5 e^{-i10\pi/4} = 32 e^{-i5\pi/2} = 32 e^{-i\pi/2}$.
4. **(a)** 1. $14 \equiv 1 \pmod{13}$, so $1^{2026} \equiv 1$.
5. **(b)** $5/2$. Highest degree terms dominate.
6. **(b)** 6. The loop iterates $3 \times 3 = 9$ times. $x \neq y$ is true for $9 - 3$ (diagonal) = 6 times.
7. **(c)** 3. $A \triangle \emptyset = (A \setminus \emptyset) \cup (\emptyset \setminus A) = A \cup \emptyset = A$. Size is 3.
8. **(b)** 80. Term is $\binom{5}{2}(2x)^3(-y)^2 = 10 \cdot 8x^3 \cdot y^2 = 80 x^3 y^2$.
9. **(b)** Reflexive, Anti-symmetric, Transitive.
10. **(c)** Must be injective. Composition of injections is injective.

** Section B (1.5 points each)

   - *Rubric:* 1.0 for reduction steps, 0.5 for distinguishing even/odd cases.

** Section C (2 points each)
1. **Induction:**
   - Base ($n=1$): $1(2) = 2$. RHS: $1(2)(3)/3 = 2$. OK.
   - Hypothesis: Assumed true for $k$.
   - Step: $\frac{k(k+1)(k+2)}{3} + (k+1)(k+2) = (k+1)(k+2)[\frac{k}{3} + 1] = (k+1)(k+2)\frac{k+3}{3}$. Matches form for $k+1$.
   - *Rubric:* 0.5 Base, 0.5 Hypothesis statement, 1.0 Algebraic step.
2. **Equivalence Relation:**
   - Reflexive: $a+b = b+a$.
   - Symmetric: $a+d=b+c \implies c+b=d+a$.
   - Transitive: Add equations $a+d=b+c$ and $c+f=d+e$.
   - Class of (0,0): $\{(a,b) : a+0 = b+0\} \implies a=b$. Diagonal line in $\mathbb{Z}^2$.
   - *Rubric:* 1.0 for Proofs (R/S/T), 1.0 for Class description.
3. **Limit:**
   - Draft: $|\frac{3n}{n+2} - 3| < \epsilon$.
   - Algebra: $|\frac{3n - 3(n+2)}{n+2}| = |\frac{-6}{n+2}| = \frac{6}{n+2}$.
   - Find N: $\frac{6}{n+2} < \frac{6}{n} < \epsilon \implies n > \frac{6}{\epsilon}$. Let $N > 6/\epsilon$.
   - *Rubric:* 1.0 for algebra simplification, 1.0 for N selection logic.
4. **Pigeonhole:**
   - Write every number $x \in A$ as $x = 2^k \cdot m$ where $m$ is odd.
   - The possible odd parts $m$ are $\{1, 3, 5, \dots, 2n-1\}$. There are exactly $n$ odd numbers.
   - We pick $n+1$ numbers, so by PHP, two numbers must have the same odd part $m$.
   - Let them be $x = 2^j m$ and $y = 2^k m$. One divides the other.
   - *Rubric:* 1.0 for $2^k m$ representation, 1.0 for applying PHP to odd parts.
5. **Complex Roots:**
   - $z^4 = 16 e^{i\pi}$.
   - Roots: $z_k = 2 e^{i(\pi + 2\pi k)/4}$ for $k=0,1,2,3$.
   - Angles: $\pi/4, 3\pi/4, 5\pi/4, 7\pi/4$.
   - Rectangular: $\sqrt{2}(1+i), \sqrt{2}(-1+i), \sqrt{2}(-1-i), \sqrt{2}(1-i)$.
   - *Rubric:* 1.0 for modulus/angles, 1.0 for rectangular forms.
6. **Derangements:**
   - Total permutations $n!$.
   - Property $P_i$: element $i$ is fixed.
   - Use $N - | \cup P_i |$. $|P_i| = (n-1)!$. Intersection of $k$ sets is $(n-k)!$.
   - Result: $D_n = n! \sum_{k=0}^n \frac{(-1)^k}{k!}$.
   - *Rubric:* 1.0 for setup of properties, 1.0 for final formula derivation.
7. **Function Proofs:**
   - (1) Injective: Assume $f(x) = f(y)$. Then $g(f(x)) = g(f(y))$. Since $g \circ f$ injective, $x=y$. Proven.
   - (2) Surjective: Let $c \in C$. Since $g \circ f$ surjective, $\exists a \in A$ s.t. $g(f(a)) = c$. Let $b = f(a)$. Then $g(b) = c$. $\exists b$, so $g$ is surjective.
   - *Rubric:* 1.0 for Injective proof, 1.0 for Surjective proof.
8. **Divisibility:**
   - $n^3 + 5n = n^3 - n + 6n$.
   - $6n$ is divisible by 6.
   - $n^3 - n = n(n-1)(n+1)$ is product of 3 consecutive integers. Divisible by 2 (at least one even) and 3 (one multiple of 3). Thus divisible by 6.
   - Sum is divisible by 6.
   - *Rubric:* 1.0 for splitting or factoring, 1.0 for arguing divisibility by 2 and 3.
