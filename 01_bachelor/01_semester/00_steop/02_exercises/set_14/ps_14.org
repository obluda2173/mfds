* Problem 1 [PARTIALLY DONE]
** Task
Find supremum and the infimum of the following sets in extended real number line.

1. A = {n/(2n^3 + 1) : n ∈ ℕ}

2. A = {(n^2)/(n + 1) : n ∈ ℕ}

3. A = {m/n : m,n ∈ ℕ, m<n}

4. A = {(m^2 + n^2)/mn : m,n ∈ ℕ}

5. A = ∪(∞, n=1)(-1 + 1/n, 1 + 1/n)

** Notes
1. A = {n/(2n^3 + 1) : n ∈ ℕ}

   n/(2n^3 + 1) = 1/(2n^2 + 1/n)

   Hence, as n->∞, 1/(2n^2 + 1/n) = 1/(∞ + 0) = 0

   Thus, supremum = 1/(2*1^3 + 1) = 1/3
          infimum = 0

2. A = {(n^2)/(n + 1) : n ∈ ℕ}

   (n^2)/(n + 1) = (n)/(1 + 1/n)

   Hence, as n->∞, 1/n -> 0. Thus, as n->∞, (n)/(1 + 1/n) -> +∞.

   Thus, supremium = +∞
         infimum = 1/1+1 = 1/2

3. A = {m/n : m,n ∈ ℕ, m<n}

4. A = {(m^2 + n^2)/mn : m,n ∈ ℕ}

5. A = ∪(∞, n=1)(-1 + 1/n, 1 + 1/n)
