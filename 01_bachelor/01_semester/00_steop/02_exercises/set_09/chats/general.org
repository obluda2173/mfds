* How would you mathematically prove or disprove the following statement:
1. There exists a bijective function f : ℚ -> ℝ.


The statement is false, there does not exist a bijective function f : ℚ -> ℝ. Here's an overview of why.

A bijective function is both injective (one-to-one) and surjective (onto). A function f : ℚ -> ℝ could be injective since every rational number could map to a unique real number. However, the same function couldn't be surjective as there are some real numbers that cannot be mapped from rational numbers.

The rationals ℚ are countably infinite, while the reals ℝ (even in an interval) are uncountably infinite. By definition, there cannot be a surjective function from a set with smaller cardinality to a set with bigger cardinality.

Cantor's Diagonal Argument is often used to prove that the real numbers are uncountable. And it further implies that there's no bijection between ℚ and ℝ.

* Prove or disprove mathematically:
2. If A,B,C are sets such that A ⊂ B ⊂ C and A and C are countable, then B is countable.

This statement can be proven true. Here is the proof:

Definition: A set is countable if it has the same cardinality as some subset of the natural numbers. In other words, if there exists a bijective function between the set and a subset of the natural numbers.

Given: A ⊂ B ⊂ C and A, C are countable.

Since C is countable, there exists a bijective function, say f, from C to a subset of natural numbers.

And since B is a subset of C, for every element b in B, there exists a natural number n such that f(b) = n. This creates a function, say g, from B to a subset of natural numbers.

Thus, B is countable.

Therefore, the statement "If A,B,C are sets such that A ⊂ B ⊂ C and A and C are countable, then B is countable", is proven true.

* Prove or disprove mathematically:
3. The set S = {root(5)/2n : n ∈ ℕ} is countable.

The set S = {root(5)/2n : n ∈ ℕ} is indeed countable.

Proof:

A set is countable if there exists a bijective (one-to-one and onto) function from the set of natural numbers (ℕ) to the set in question.

In our case, we can see that the function f: ℕ -> S defined by f(n) = root(5)/2n, for all n in ℕ, is a bijection:

(i) f is injective (one-to-one): Assume f(n) = f(m) for some natural numbers n and m. Then by definition of f, root(5)/2n = root(5)/2m. If we clear the fractions we get 2m = 2n and hence m = n.

(ii) f is surjective (onto): For every element a in S, there exists n in ℕ, such that f(n) = a by the definition of the set S.

Hence, by definition, the set S is countable.
* Prove or disprove mathematically:
4. There exists a countable subset of the set of irrational numbers.

Prove:

We know the set of algebraic numbers is countable and these numbers include both rational and irrational numbers. Because algebraic numbers are countable, there exists a one-to-one correspondence to the set of natural numbers.

Let's construct a countable subset of irrational numbers as follows:

Consider the algebraic numbers that are also square root of all positive integers. We can write these numbers as ±√n for n = 1, 2, 3, .... Clearly, all such numbers are not rational (except for perfect squares, but we'll discard those). Since we can enumerate all positive integers, we can also enumerate all these numbers, so this represents a countable set. It's a subset of all algebraic numbers, which is in turn a subset of all numbers.

Thus, we have shown that there exists a countable subset of the set of irrational numbers, namely the set {±√k | k is an integer, and √k is irrational}.

* Give another proof

Sure, here is another way to prove it:

We are asked to find a countable subset of irrational numbers. A simple example is the set formed by the square roots of the prime numbers.

We note that each prime number is unique (by definition), and the square root of a prime number is irrational (since it cannot be expressed as a ratio of two integers, except in trivial cases which are the primes themselves). Moreover, the prime numbers are countable, since we can list them as p1, p2, p3, p4, ..., where pn is the nth prime.

So, for each pn, we can associate the number √pn, which is an irrational number. Therefore, the subset of square roots of primes is countable, giving us a countable subset of the set of irrational numbers.
