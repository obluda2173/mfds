* Transcript Cleaning Prompt

Clean the following lecture transcript produced by OpenAI Whisper from a
recorded university lecture. Apply the following corrections:

** 1. Technical terminology repair

Fix systematic mistranscriptions of domain-specific terms. Examples from
this transcript include:
- "hold" / "holds" → "halt" / "halts" (in the context of algorithm termination)
- "chase" → "j" (the loop variable)
- "Jake" → "j"
- Use the surrounding mathematical context to identify and correct similar
  errors throughout.

** 2. Remove non-content passages

Strip out classroom management noise that carries no intellectual content:
- Equipment setup (projector, beamer, blackboard adjustments)
- Filler phrases ("just a second", "let me see", "come on")
- Dialogue about lights, seating, logistics
- Greetings and sign-offs unless they contain course-relevant information
  (e.g. what will be covered next week)

Preserve any meta-commentary that *is* pedagogically relevant, such as
the lecturer signalling what is important, what will be on the exam, or
what was covered in a previous session.

** 3. Sentence-level cleanup

- Merge sentence fragments that Whisper split incorrectly into coherent
  sentences.
- Remove false starts and verbal repetitions (e.g. "So we define some
  property. Then we check. Check. That's the first step. Check that or
  proof, actually proof or show." → clean single formulation).
- Do *not* paraphrase or add content. Stay faithful to what the lecturer
  actually said; only remove noise and repair transcription errors.

** 4. Output format

Return the cleaned transcript as a continuous text, preserving the
original ordering. Insert blank lines between major topic transitions to
make the structure visible.
