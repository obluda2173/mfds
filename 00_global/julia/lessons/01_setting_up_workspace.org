* Step 1: The Essentials
1. **Download VS Code:** Get it from [[https://code.visualstudio.com/][code.visualstudio.com]].
2. **Install the Julia Extension:**
   - Open VS Code.
   - Click the generic "Tetris block" icon on the left sidebar (Extensions).
   - Search for =Julia=.
   - Click *Install* (the one by "Julia Language Support").

/Tip: You might also want to install the "Python" extension if you plan to use Python later, or "Markdown All in One" for better note-taking./

* Step 2: The Razyeb Stuff
You are going to stare at this screen for hours. Let's make it look good.

** VS Code Pets 🐈
/Getting better at Julia/ can be lonely especially if you have no friends. Let's get a coding buddy.
1. Go to Extensions.
2. Search for =vscode-pets=.
3. Install it.
4. Open the Command Palette (=Ctrl + Shift + P= or =Cmd + Shift + P= on Mac).
5. Type =Pets: Start pet coding session=.
6. A small panel will open with a cat (or dog/snake/clippy). You can throw a ball at it while you think about math!

** Themes & Icons
Don't stick with the default look.
- *Themes:* Search extensions for "Dracula", "One Dark Pro", or "GitHub Theme".
- *Icons:* Search for "Material Icon Theme". It changes the boring file icons into pretty logos (Julia logo for .jl files, etc.).

* Step 3: Mastering the REPL
Open a Julia file or start the terminal. Type =julia= to start the engine.

The *REPL* (Read-Eval-Print Loop) is where you talk to Julia. It has "moods" (modes) that change based on the first key you press.

| Key    | Color      | Name       | Function                                                    |
| :---   | :---       | :---       | :---                                                        |
| (None) | **Green**  | **Julian** | Standard coding (math, variables, functions).               |
| =?=    | **Yellow** | **Help**   | Search documentation. Type =?sin= to see how sine works.    |
| =]=    | **Blue**   | **Pkg**    | Package Manager (installing stuff).                         |
| =;=    | **Red**    | **Shell**  | System commands. Use =ls= or =mkdir= without leaving Julia. |

*How to exit a mode?* Just press =Backspace= on an empty line to return to Green.

* Cheatsheet: Keyboard Shortcuts
Memorize these three. They will save you hours of life.

1. *Run Code:* =Shift + Enter=
   /Runs the current line and moves to the next one./
2. *Comment/Uncomment:* =Ctrl + /= (or =Cmd + /=)
   /Turns code into text so the computer ignores it./
3. *Command Palette:* =Ctrl + Shift + P= (or =Cmd + Shift + P=)
   /The search bar for VS Code features. If you can't find a button, search for it here./
