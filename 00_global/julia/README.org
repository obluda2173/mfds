* Why Julia?
Julia is a relatively new high-level language. While this historically meant there were fewer tutorials, it is the superior choice for MFDS for three reasons:
1. *Speed:* It approaches the speed of C/C++, meaning your numerical integrations and matrix operations finish instantly compared to Python.
2. *Math-Friendly Syntax:* The code looks like the math on your paper. You can literally use symbols like $\in$, $\le$, or $\sigma$ as variable names.
3. *Multiple Dispatch:* It handles mathematical types (like complex numbers) much more naturally than object-oriented languages.

* The Workflow: Why "The Cloud" Sucks
You might be tempted to run Julia in a browser (Jupyter Notebooks, or if you are /netakusya/ Binder, Colab). *Don't do it.*

** Why Browser-based coding fails:
1. *State Issues (The Grading Trap)* Browser notebooks allow you to run code out of order, hiding bugs.

   *Student Reality*: You submit code that works on your screen, but crashes when the professor runs it from top-to-bottom. Result: You lose marks.

2. *Resource Limits (The "Timeout" Risk)* Free cloud servers are weak and disconnect easily

   *Student Reality*: As your math models get complex, cloud notebooks will crash or time out in the middle of a calculation, forcing you to restart hours of work.

3. *No Persistence (The File Nightmare)* Browsers isolate you from your file system.

   *Student Reality*: You have to re-upload your data every time you open the page. Coding locally means your datasets, plots, and versions are saved permanently on your drive.

4. *Offline Independence* Browser coding requires stable internet.

   *Student Reality*: A local setup lets you code on the train, in a cafe, or during a WiFi outage.

** The Solution: VS Code
If you are begginer, I strictly recommend using *Visual Studio Code* installed locally. It offers a full IDE experience with a variable explorer, plot viewer, and instant feedback, without the chaos of browser notebooks.

* Installation Guide
Follow these steps exactly to get a working environment.

** Step 1: Install Julia (The Modern Way)
We use `juliaup`, the official version manager and updater.

*** Windows
Open PowerShell (Win+R, type `powershell`) and run:
#+begin_src powershell
winget install julia -s msstore
#+end_src

*** Mac / Linux
Open your terminal and run:
#+begin_src bash
curl -fsSL https://install.julialang.org | sh
#+end_src

/Note: Restart your terminal after installation./

** Step 2: Install VS Code
Download it here: [[https://code.visualstudio.com/][code.visualstudio.com]]

** Step 3: Connect them
1. Open VS Code.
2. Go to Extensions (Ctrl+Shift+X).
3. Search for "Julia" and install the official extension by Julia Graphics.
