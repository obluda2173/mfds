** Counting Infinities
- Hilbert's Hotel experiment
  - Assume every room at the infinite hotel is occupied, and there is no empty room. With most hotels this would mean that if someone else arrives at the hotel, they will not be given a room. However! if for n ∈ ℕ, the patron in room 1 moves to room n + 1, then nobody is left without a room and suddenly room 1 is completely open. We created room out of nothing?

  - Now lets imagine two people arrived to the hotel. We can also accommodate them with the same logic. Now everyone move from room n to room n + 2. This leaves rooms 1 and 2 open for new customers.

  - What if, however, we have infinitely many people lined up wanting a room? Can we accommodate all of them? Yes! We still can. Just leave the person in room n and move to room 2n. Then all of the odd numbered rooms are vacant and the infinite line of people can take these rooms.
- Specific Sets

  There are the same number of natural numbers as there are natural numbers larger than 1 (|ℕ| = |{2,3,4,...}|). What is the bijection that shows this? Let

       f : ℕ -> {2,3,4,...} where f(n) = n + 1.

  In other words this is the pairing

       1<->2 2<->3 3<->4

  *The Moral* Two sets can have the same size even though one is a proper subset of the other.

  - Integers
    One of the most popular ways to prove the remarkable fact that |ℤ| = |ℕ| is:

    Imagine integer set:

    ℤ = {..., -2, -1, 0, 1, 2, ...}

    Thus, we can reshape it into:

    ℤ = {0, 1, -1, 2, -2, 3, -3, ...}

    This way we can have a clean start of counting, propper incremention technique, that does not miss any elements of the set. Thus ℤ is countable.

  - Rationals
    Construc the following table:

    1/1  2/1  3/1  4/1  5/1 ...

    1/2  2/2  3/2  4/2  5/2

    1/3  2/3  3/3  4/3  5/3

    1/4  2/4  3/4  4/4  5/4

    1/5  2/5  3/5  4/5  5/5

    ...

    And now, starting from 1/1, we will go diagonally, collecting all of the values, so it would look like:

    {1/1, 2/1, 1/2, 3/1, 2/2, 1/3, ...}

    With this, it is the case that |ℕ| = |ℤ| = |ℚ|

* Cantor-Schröder-Bernstein Theorem
#+begin_quote
*Theorem.*
If there is an injection f : A -> B and g B -> A, then there exists a bijection h : A -> B.

In cardinality language, if |A| <= |B| and |B| <= |A|, then |A| = |B|.
#+end_quote

** Theorem proof
*** General plan
- Set up and definitions
- Lemma 1 - Well definedness of the second case
- Lemma 2 - The equivalence of b ∈ f(A*) <--> g(b) ∈ A*
- Corollary - Disjointness of the two image sets used by h

*** Detailed plan
- High-level road-map
  1. *Setup and data* - inputs and what we will use.
  2. *Isolate forward starting points* - build A0, A1, A2, ... A*. Purpose: find the part of A you will map with f.
  3. *Define a candidate bjection h* - rule that depends on membership in A*.
  4. *Show the rule is well defined* = show that second case actually chooses a unique element.
  5. *Prove the key equivalence* - b ∈ f(A*) <--> g(b) ∈ A*.
  6. *Deduce the partition property* -
  7. *Prove injectivity* - no two a's map to the same b.
  8. *PRove surjectivity* - every b ∈ B is hit by some a.
  9. *Conclude bijection* - gather pieces.

- *Block 1 - Setup and data*
  - *Goal:* establish notations and assumptions.

  - *Given:* sets A and B, and bijections f : A -> B and g : B -> A.

  - *Why necessary:* every other step is build on f and g functions injectivity.

  - *What is permits:* defining images f(X), g(Y) and taking preimages uniquely when maps are injective.

- *Block 2 - Isolate forward starting points*
  - *Definitions:*
    - A0 = A \ g(B)  (those a with no incoming g arrow)

    - An = f(g(An-1))
      or
      An+1 = f(g(An))

    - A* = U(n>=0)(An)

  - *Goal:* identify exaclty those a ∈ that lie on components that start in A (no predecessor).

  - *Guaranties:* an explicit subset A* ∈ A that collects all A-nodes that are reachable from A0 by alternating f,g.

- *Block 3 - Define h*
  - *Rule:* for a ∈ A
    - if a ∈ A*, h(a) = f(a)
    - if a ∉ A*, h(a) = b, such that g(b) = a

  - *Goal:* give a single function that will be proven bijective.

  - *Immediate questions to attack next:* is the second case well defined? and do the two pieces collide?

- *Block 4 - Well-definedness of second case*
  - *Claim:* if a ∉ A*, then a ∈ g(B), so there exist a unique b with g(b) = a.

  - *Uses:* A0 = A \ g(B) and A0 ⊆ A*.

  - *Why necessary:* otherwise rule is ill-specified.

- *Block 5 - Prove key equivalence*
  - *Statement:* for any b ∈ B:

        b ∈ f(A*) <--> g(b) ∈ A*

  - *Why this matters:* it ties two parts of the definition of h together; it says "b is in the forward image of A* exactly when its g-arrows lands inside A*".

  - *Uses:* the recursive definition of An and injectivity of g (to move from g(f(a)) = g(b) to f(a) = b).

- *Block 6 - Partition/separation corollary*
  - *Statement:* f(A*) and B \ f(A*) are exactly the images of the two branches of h. Concretely:

    - if a ∈ A*, h(a) ∈ f(A*)
    - if a ∉ A*, h(a) ∈ B \ f(A*)

  - *Why this mattes:* ensures no collision between a mapped by f and an a' mapped by g^-1. This is a structural reason injectivity works.

  - *Uses:* equivalence from Block 5.

- *Block 7 - Injectivity on h*
  - *Strategy:* split into three exhaustive cases:
    - both from A* -> reduce to unjectivity of f.
    - both not from A* -> reduce to injectivity of g.
    - one in A* -> impossible because of the separation from Block 6.

  - *Why it matters:* shows no 2 different a's map to the same b.

  - *Uses:* Blocks 4-6 and injectivity of f,g.

- *Block 8 - Surjectivity on h*
  - *Strategy:* take any b ∈ B. Two cases by Block 5:
    - if b ∈ f(A*), then pick that a ∈ A* with f(a) = b and h(a) = b

    - if b ∉ f(A*), then g(B) ∉ A*; by Block 4 the rule picks h(g(b)) = b

  - *Why it matters:* ensures every b is hit.

  - *Uses:* Block 4-5 and definition of h.

- *Block 9 - Conclusion*
  - *Combining:* well-defined (Block 4) + injective (Block 6) + surjective (Block 7) --> h is a bijection A -> B.

*** Block-dependency graph
- Setup → (defines f,g).

- Setup → Block 2 (builds A*).

- Block 2 → Block 3 (definition of

- Block 2 + Setup → Block 4 (well-definedness).

- Block 2 → Block 5 (equivalence).

- Block 5 → Block 6 (separation).

- Block 6 + Setup → Block 7 (injectivity).

- Block 4 + Block 5 → Block 8 (surjectivity).

- Blocks 4,7,8 → Block 9 (conclusion).

*** Proof
- *Set up and definitions*

  Let A, B be sets with injective functions f : A -> B and g : B -> A.

  *Define.*

  - A0 = A \ g(B)

    A0 is the set of elements of A that are not mapped by any element of B with g.
    or
    A0 are the A0 nodes with no incoming arrows from B.

  - For each n >= 0, define

    An+1 = g(f(An))

    First apply f to the whole set An, you get subset of B. Then apply g on that subset, and get subset of A. So A1 = g(f(A0)) and etc.

  - Let A* = U(n>=0)(An)

    A* is the union of these iterated images. A* is a subset of A, containing elements that can be reached by using above An+1 = g(f(An)) method.

  And lastly, we define the candidate bijection function h : A -> B:

        h(a) = / f(a) for a ∈ A*,
               \ b ∈ B, g(b) for a ∉ A*.

  We need to justify two things about this definition:
  1. that there exist such unique b ∈ B (second case)
  2. h is a bijection

- *Lemma 1 - Well definedness of the second case*

  *Claim.* If a ∉ A*, then ∃ b ∈ B, g(b) = a.

  *Proof.* By definition A0 = A \ g(B),

  - a ∈ A0 iff a ∉ g(B).
  - a ∉ A0 iff a ∈ g(B).

  Since A0 ⊆ A, and a ∉ A*, a ∉ A0. Thus, a ∈ g(B).

  Since g is injective, the preimage of a, b is unique. Proved.

- *Lemma 2 - The equivalence of b ∈ f(A*) <--> g(b) ∈ A**

  *Claim.* For every b ∈ B:

  b ∈ f(A*)  and  g(b) ∈ A*.

  *Proof.*
  - (-->) Suppose b ∈ f(A*), then b = f(a) for some a ∈ A*. Than a ∈ A* means a ∈ An for some n. By definition, An+1 = gf(An), so gf(An) ⊆ An+1 ⊆ A*. In particular g(f(a)) ∈ A*. But g(f(a)) = g(b). Hence g(b) ∈ A*.

  - (<--) Suppose g(b) ∈ A*, then g(b) ∈ Am for some m >= 0. For m = 0, we have A0 = A \ g(B), but g(b) ∈ g(B), so m can not be 0. Hence m >= 0. By the recursive definition, for m >= 1 there exists some a ∈ Am-1 with

        g(f(a)) = (an element of g(f(Am-1)) = Am)

    Because g(b) ∈ Am = g(f(Am-1)), there exists a ∈ Am-1 with g(f(a)) = g(b). But g is injective, so f(a) = b. Therefore, b ∈ f(Am-1) ⊆ f(A*). Proved.

- *Corollary - Disjointness of the two image sets used by h*

  From the equivalence, we get important separation:

  - If a ∈ A*, then h(a) = f(a) ∈ f(A*)

  - If a ∉ A*, then h(a) = b, with g(b) = a. By the equivalence, g(b) = a ∉ A* implies b ∉ f(A*). Hence in this case h(a) ∈ B \ f(A*).
