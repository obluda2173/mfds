* Problem 1 [DONE]
** Task
Prove
1. (A \ B) \ C = (A \ C) \ (B \ C)
2. ∩(x ∈ R)[3 - x^2, 5 + x^2] = [3, 5]

** Solution
1. (A \ B) \ C = (A \ C) \ (B \ C)
   Not quite sure of how to prove it.
   - Review
     Let x ∈ (A \ B) \ C.
     Thus, (x ∈ A) ∧ (x ∉ B) ∧ (x ∉ C)

     Let x ∈ (A \ C) \ (B \ C).
     Thus, (x ∈ A) ∧ (x ∉ C) ∧ (x ∉ (B \ C))
          x ∉ (B \ C)
          (x ∉ B) ∨ (x ∈ C)
          Since (x ∉ C), x (x ∉ B)
          Thus, (x ∈ A) ∧ (x ∉ B) ∧ (x ∉ C)

     Hence, (A \ B) \ C <--> (A \ C) \ (B \ C)

2. ∩(x ∈ R)[3 - x^2, 5 + x^2] = [3, 5]
   - Original
     As x increases, the next range also increases in both sides:

     x = 0 --> [3 - 0, 5 + 0] = [3, 5]
     x = 1 --> [3 - 1, 5 + 1] = [2, 6]
     x = 2 --> [3 - 4, 5 + 4] = [-1, 9]
     ...

     Since there is an x^2, each range would be greater or equal to [3, 5]. Meaning that the intersection of all ranges would be [3, 5].

   - Review
     Since x^2 >= 0, [3, 5] ⊆ ∩(x ∈ R)[3 - x^2, 5 + x^2]

     Take arbitrary y, s.t. y ∈ [3, 5]. Hence,

     ∀x : 3 - x^2 <= 3 <= y <= 5 <= 5 + x^2

     Hence,
     ∩(x ∈ R)[3 - x^2, 5 + x^2] = [3, 5]

* Problem 4 [DONE]
** Task
Let P,Q, and R be the propositions
• P: You have the flu.
• Q: You miss the final examination.
• R: You pass the course.

Express each of these propositions as an English sentence.
1. P => Q
2. ¬Q <=> R
3. Q => ¬R
4. (P => ¬R) ∨ (Q => ¬R)

** Solution
1. P => Q

   If you have a flu, you would miss the final examination.

2. ¬Q <=> R

   You would pass examination, if and only if you would not miss the course.

3. Q => ¬R

   If you miss the final examination, then you would not pass the course.

4. (P => ¬R) ∨ (Q => ¬R)

   If you have the flu or you miss the final examination, then you would not pass the course.

* Problem 5 [DONE]
** Task
Five friends have access to a chat room. Is it possible to determine who is chatting if the following information is known?

- Either Kevin or Heather, or both, are chatting.
- Either Randy or Vijay, but not both, are chatting.
- If Abby is chatting, so is Randy.
- Vijay and Kevin are either both chatting or neither is.
- If Heather is chatting, then so are Abby and Kevin.

Explain your reasoning.

** Solution
- K ∨ H
- (R ∨ V) ∧ ¬(R ∧ V)
- A --> R
- V <--> K
- H --> (A ∧ K)

Starting with V <--> K, since it is the only "fact" and not a "link".

Thus, we have two options:

- If V = False, K = False
  Thus,
  - K ∨ H: nothing
  - (R ∨ V) ∧ ¬(R ∧ V): nothing
  - A --> R: nothing
  - H --> (A ∧ K): nothing

  - If H = False
    Then,
    - K ∨ H: H = False
    - (R ∨ V) ∧ ¬(R ∧ V): contradiction, since they can not not chat together
    - A --> R: since A = False, R = False
    - H --> (A ∧ K): A = False, K = False

  - If H = True
    Then,
    - K ∨ H: H = True
    - (R ∨ V) ∧ ¬(R ∧ V):
    - A --> R: R = True
    - H --> (A ∧ K): A = True, K = True. Contradiction.

- If V = True, K = True
  Thus,
  - K ∨ H: nothing
  - (R ∨ V) ∧ ¬(R ∧ V): R = False
  - A --> R: nothing
  - H --> (A ∧ K): nothing

  - If H = False
    Then,
    - K ∨ H: H = False
    - (R ∨ V) ∧ ¬(R ∧ V): R = False
    - A --> R: since A = False, R = False
    - H --> (A ∧ K): A = False, K = False

    Thus, only K and V are chatting

  - If H = True
    Then,
    - K ∨ H: H = False
    - (R ∨ V) ∧ ¬(R ∧ V): V = False. This contradicts initial H = True
    - A --> R: since A = True, R = True
    - H --> (A ∧ K): A = True, K = True

    Contradiction.

* Problem 7 [DONE]
** Task
Express each of these statements using quantifiers.
Then form the negation of the statement and simplify it.
Next, express the negation in simple English.

1. All dogs have fleas.
2. There is a horse that can add.
3. Every koala can climb.
4. No monkey can speak French.
5. There exists a pig that can swim and catch fish.

** Solution
1. All dogs have fleas.

   - ∀x(Dogs(x) --> HasFleas(x))

   - ¬(∀x(Dogs(x) --> HasFleas(x)))
     ∃x¬(Dogs(x) --> HasFleas(x))
     ∃x¬(¬Dogs(x) ∨ HasFleas(x))
     ∃x(Dogs(x) ∧ ¬HasFleas(x))

   - There exist a dog with no fleas.

2. There is a horse that can add.

   - ∃x(Horse(x) ∧ CanAdd(x))

   - ¬(∃x(Horse(x) ∧ CanAdd(x)))
     ∀x¬(Horse(x) ∧ CanAdd(x)))
     ∀x(¬Horse(x) ∨ ¬CanAdd(x)))
     ∀x(Horse(x) --> ¬CanAdd(x)))

   - All horses can not add.

3. Every koala can climb.

   - ∀x(Koala(x) --> CanClimb(x))

   - ¬(∀x(Koala(x) --> CanClimb(x)))
     ∃x¬(Koala(x) --> CanClimb(x))
     ∃x¬(¬Koala(x) ∨ CanClimb(x))
     ∃x(Koala(x) ∧ ¬CanClimb(x))

   - There exists a koala that can not climb.

4. No monkey can speak Frenchu.

   - ∀x(Monkey(x) --> ¬SpeakFrench(x))

   - ¬(∀x(Monkey(x) --> ¬SpeakFrench(x)))
     ∃x¬(Monkey(x) --> ¬SpeakFrench(x))
     ∃x¬(¬Monkey(x) ∨ ¬SpeakFrench(x))
     ∃x(Monkey(x) ∧ SpeakFrench(x))

   - There exists a monkey that speak French.

5. There exists a pig that can swim and catch fish.

   - ∃x(Pig(x) ∧ CanSwimm(x) ∧ CanCatchFish(x))

   - ¬(∃x(Pig(x) ∧ CanSwimm(x) ∧ CanCatchFish(x)))
     ∀x¬(Pig(x) ∧ CanSwimm(x) ∧ CanCatchFish(x))
     ∀x(¬Pig(x) ∨ ¬CanSwimm(x) ∨ ¬CanCatchFish(x))

   - All pigs can either swimm or catch fish exclusively.

* Problem 8 [DONE]
** Task
1. Show that ∀x P(x) ∨ ∀x Q(x) and ∀x (P(x) ∨ Q(x)) are not logically equivalent.
2. Show that ∃x P(x) ∨ ∃x Q(x) and ∃x (P(x) ∨ Q(x)) are logically equivalent.

** Solution
1. ∀x P(x) ∨ ∀x Q(x) != ∀x (P(x) ∨ Q(x))

   D = {a,b}
   - P(a) = true; P(b) = false
   - Q(a) = false; Q(b) = true

   Thus,
   - ∀x (P(x) ∨ Q(x)) is *true* because:
     - at x = a, P(a) ∨ Q(a) = true ∨ false = true
     - at x = b, P(b) ∨ Q(b) = false ∨ true = true

   - ∀x P(x) ∨ ∀x Q(x) is *false* because:
     - P(x) is false, since P(b) is false
     - Q(x) is false, since Q(a) is false
     Thus, ∀x P(x) ∨ ∀x Q(x) is false.

2. ∃x P(x) ∨ ∃x Q(x) <--> ∃x (P(x) ∨ Q(x))

   ∃x (P(x) ∨ Q(x)) = ¬(¬(∃x (P(x) ∨ Q(x))))
                    = ¬(∀x¬(P(x) ∨ Q(x)))
                    = ¬(∀x(¬P(x) ∧ ¬Q(x)))
                    = ¬(∀x¬P(x) ∧ ∀x¬Q(x))
                    = ∃xP(x) ∨ ∃xQ(x)
