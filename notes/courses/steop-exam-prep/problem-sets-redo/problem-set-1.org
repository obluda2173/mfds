* Problem 1 [DONE]
** Task
Write each of the following sets by listing their elements between braces
1. {x ∈ ℤ : |2x| < 5}
2. {x ∈ ℝ : x^3 + 5x^2 = -6x}
3. {x ∈ ℝ : sin(Pi*x) = 0}
4. {5x : x ∈ ℤ, |2x| <= 8}

** Solution
1. {x ∈ ℤ : |2x| < 5}

   {x ∈ ℤ : |2x| < 5} = {-2,-1,0,1,2}

2. {x ∈ ℝ : x^3 + 5x^2 = -6x}

   {x ∈ ℝ : x^3 + 5x^2 = -6x} = {-3,-2,0}

   ^3 + 5x^2 + 6x = 0
   x(x^2 + 5x + 6) = 0  --> x1 = 0
   x^2 + 5x + 6 = 0

   Thus,
   x = (-5 ± sqrt{25 - 4 * 6})/(2)
   x = (-5 ± sqrt{1})/(2)
   x = (-5 ± 1)/(2)

   Hence,
   x1 = 0
   x2 = (-5 + 1)/(2) = -2
   x3 = (-5 - 1)/(2) = -3

3. {x ∈ ℝ : sin(Pi*x) = 0}

   {x ∈ ℝ : sin(Pi*x) = 0} = ℤ

4. {5x : x ∈ ℤ, |2x| <= 8}

   {5x : x ∈ ℤ, |2x| <= 8} =

   x = {-4, -3, -2, -1, 0, 1, 2, 3, 4}

   Thus,
   {-20, -15, -10, -5, 0, 5, 10, 15, 20}

* Problem 2 [DONE]
** Task
Write each of the following sets in set-builder notation
1. {2, 4, 8, 16, 32, 64, ...}
2. {-5, -4, -3, -2, -1, , 0, 1}
3. {..., 1/27, 1/9, 1/3, 1, 3, 9, 27}
4. {3, 6, 11, 18, 27, 38, ...}

** Solution
1. {2, 4, 8, 16, 32, 64, ...}

   {2^x : x ∈ ℕ, x >= 2} = {2, 4, 8, 16, 32, 64, ...}

   Correct: {2^x : x ∈ ℕ, x >= 1}

2. {-5, -4, -3, -2, -1, , 0, 1}

   {x : x ∈ ℤ : -5 <= x <= 1} = {-5, -4, -3, -2, -1, , 0, 1}

3. {..., 1/27, 1/9, 1/3, 1, 3, 9, 27}

   {3^x : x ∈ ℤ, x <= 3} = {..., 1/27, 1/9, 1/3, 1, 3, 9, 27}

1. {3, 6, 11, 18, 27, 38, ...}

   {x^2 + 2 : x ∈ ℕ} = {3, 6, 11, 18, 27, 38, ...}
