* Problem 1 [DONE]
** Task
Let x and y be nonnegative numbers. Prove that if x <= y, then sqrt{x} <= sqrt{y}.

** Solution
#+begin_quote
*Proposition.* x,y >= 0, if x <= y, then sqrt{x} <= sqrt{y}.

*Proof.* Assume x <= y with x,y >= 0.

Square both sides of sqrt{x} <= sqrt{y}:

sqrt{x}^2 = x
sqrt{y}^2 = y

Squaring is allowed since both sides are non-negative.

Thus, from x <= y, we get:

sqrt{x}^2 <= sqrt{y}^2

Since the square function is increasing on [0, ∞), this is equivalent to:

sqrt{x} <= sqrt{y}
#+end_quote

* Problem 2 [DONE]
** Task
For n ∊ ℕ prove that 4|(1 + (-1)^n * (2n - 1)).

** Solution
#+begin_quote
*Proposition.* For n ∈ ℕ, 4|(1 + (-1)^n * (2n - 1)) <--> (1 + (-1)^n * (2n - 1)) ≡ 0 (mod 4).

*Proof.*

(1 + (-1)^n * (2n - 1))

n = 1 : (1 + (-1)^1 * (2*1 - 1)) = (1 - 1(2 - 1)) = 0
n = 2 : (1 + (-1)^2 * (2*2 - 1)) = (1 + 1(4 - 1)) = 4
n = 3 : (1 + (-1)^3 * (2*3 - 1)) = (1 - 1(6 - 1)) = -4
n = 4 : (1 + (-1)^4 * (2*4 - 1)) = (1 + 1(8 - 1)) = 8
n = 5 : (1 + (-1)^5 * (2*5 - 1)) = (1 - 1(10 - 1)) = -8

Let n = 2k for k ∈ ℕ.
        (1 + (-1)^(2k) * (2*2k - 1)) = (1 + 1 * (4k - 1))
                                     = 1 + 4k - 1
                                     = 4k

Let n = 2k + 1 for k ∈ ℕ.
        (1 + (-1)^(2k+1) * (2*(2k+1) - 1)) = (1 - 1 * (4k + 2 - 1))
                                           = 1 - 4k - 1
                                           = -4k

*Conclusion.* In both cases, expression is a multiple of 4 for every n ∈ ℕ.
#+end_quote
