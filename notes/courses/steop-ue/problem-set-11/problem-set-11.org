** Task
Find a quadratic polynomial with integer coefficients that has one root equal to 3 - 7^1/2.

** Notes
x = (-b±(root(b^2 - 4*ac)))/(2a)

x1 = 3 - 7^1/2

Since polynomial has integer coefficients, the conjugate root will also be root:

x2 = 3 + 7^1/2

(x-x1)(x-x2) = 0

(x - (3 - 7^1/2))(x - (3 + 7^1/2)) = 0

x^2 - x(3 + 7^1/2) - x(3 - 7^1/2) + (3 - 7^1/2)(3 + 7^1/2) = 0

x^2 + x(-3 - 7^1/2 - 3 + 7^1/2) + (3^2 - 7) = 0

x^2 - 6x + 2 = 0
